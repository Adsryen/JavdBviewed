# 常用接口速查（本项目相关 · Jellyfin 12.0.0）

> 文档目录：`jellyfin-api-12.0.0`
> 官方来源：https://api.jellyfin.org/openapi/jellyfin-openapi-stable.json

## GET /System/Info/Public

公开服务器信息。见 `系统管理/系统信息-System.md` → GetPublicSystemInfo。

## GET /System/Info

完整服务器信息。见 `系统管理/系统信息-System.md` → GetSystemInfo。

## GET /Items

**产品查影视库主接口**（tag: Library）。

| 参数 | 值 | 说明 |
| --- | --- | --- |
| Recursive | true | 递归 |
| IncludeItemTypes | Movie,Series,Episode | 电影/剧/集 |
| Fields | Path,ProviderIds,Overview,ProductionYear,UserData,PrimaryImageAspectRatio,ImageTags | 元数据与状态 |
| StartIndex / Limit | 分页 | 大库必用 |
| SearchTerm | 可选 | 关键字 |
| api_key / X-Emby-Token | *** | Token |

见 `媒体库与条目/媒体库与条目-Library.md` → GetItems。

## GET /Items/{itemId}

单条目详情。见同文件 → GetItem。

## GET /UserViews

用户媒体库视图。见 `媒体库与条目/用户视图-UserView.md`。

## GET /Library/MediaFolders

媒体文件夹。见 `媒体库与条目/媒体库与条目-Library.md`。

## GET /Search/Hints

搜索提示。见 `媒体库与条目/搜索-Search.md`。

## POST /Users/AuthenticateByName

登录换 Token。见 `接入认证/认证-Authentication.md`。
