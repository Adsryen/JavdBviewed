快速开始

  • 打开控制面板：点击扩展图标后进入“控制面板”。
  • 首次配置：在“设置”中完成 WebDAV 同步、功能增强、115 网盘、隐私保护等开关与参数配置。
  • 立即体验：在“番号库/演员库/新作”中浏览、收藏，并管理你的记录。

数据管理

  • 番号库：本地记录库，支持搜索、标签筛选、排序、分组、批量标记、编辑、删除。
  • 演员库：同步 JavDB 收藏演员，支持别名/分类/拖拽排序，并支持分页与卡片大小调节。
  • 新作：订阅演员并自动发现其新作，提供过滤、状态联动、清理已读与分页浏览。
  • 数据导入/导出：支持本地 JSON 备份与还原；可通过 WebDAV 在多设备间备份与恢复。
  • 统计与概览：实时统计“已观看/已想看/想再看”等数量，首页图表与概览模块展示。

账户信息

  • 账户登录：显示并管理你的 JavDB 账户信息（邮箱/用户名/用户类型等）。
  • 仅用于同步：账户凭据仅在本地用于与 JavDB 的数据同步，安全存储。

数据同步（JavDB ↔ 本地）

  • 同步全部：从 JavDB 获取“已观看 + 想看”的所有记录到本地。
  • 分类同步：可单独拉取“想看”或“已看”的记录，按需更新。
  • 收藏演员：支持同步收藏演员，并可单独刷新演员维度信息。
  • 进度与取消：显示“获取列表/拉取详情”两阶段进度，支持随时取消。

WebDAV 备份与恢复

  • 自动/手动：支持手动上传/下载；可按需开启自动备份，保持多设备一致。
  • 服务兼容：兼容 Nextcloud/Seafile/TeraCloud/Yandex 等主流 WebDAV 服务。
  • 恢复与合并：提供“快速替换/合并/专家”三种模式，支持智能合并、策略选择与差异分析。
  • 可选内容：可分类型备份“扩展设置/观看记录/账户信息/演员库/日志/新作”等数据。

115 网盘集成

  • 一键推送下载：在详情页可将磁力链接一键推送至 115 离线下载。
  • 验证码处理：自动处理验证流程，失败可重试，成功后可自动联动标记状态。
  • 配额侧边栏：启用 115-V2 后，可在侧栏显示网盘配额（总量/已用/剩余）。

Emby 增强

  • 信息联动：在 Emby 页面增强展示与跳转信息（与番号、演员信息联动等）。

显示与内容过滤

  • 列表隐藏：访问 JavDB 时可自动隐藏“已看/已阅/VR”等条目，精简列表。
  • 样式自定义：可调整标记颜色、展示位置与样式，并可控制细节与密度。

搜索引擎跳转

  • 自定义导航：在番号处点击跳转预设的外部搜索站点，使用 {{ID}} 作为占位符。
  • 图标与顺序：可设置显示图标与顺序，便于快速分流检索。

功能增强

  • 磁链聚合：自动聚合站内外磁链资源并去重（参见“功能增强-磁链聚合”）。
  • 快速复制：一键复制番号/标题/磁链等，提升整理效率。
  • 键位优化：支持键盘快捷操作（上一页/下一页等）。
  • 详情增强：在视频详情页提供标记、跳转与辅助信息展示。

隐私保护

  • 锁屏模式：一键模糊敏感区域；切换标签后自动恢复隐藏效果。
  • 隐私设置：在“设置 > 隐私保护”中自定义策略与强度。

快捷键

  • Ctrl + Shift + M：快速标记当前条目为“已观看”。
  • Ctrl + Shift + W：快速标记当前条目为“想看”。
  • 多选操作：支持 Ctrl/Shift 组合多选；点击状态标签可快速切换。

日志与诊断

  • 运行日志：记录操作、警告与错误，支持按等级筛选、清空、导出。
  • 网络测试：测试 WebDAV 连通性、响应时间与同步性能，辅助排障。

高级工具

  • 数据结构校验：自动检查并修复历史数据格式问题，保持一致性。
  • 数据迁移：从旧版本升级后自动迁移数据结构，可显示进度。
  • JSON 编辑：为高级用户提供原始 JSON 查看与编辑（请谨慎操作）。

常见问题

  • 数据丢失：可通过 WebDAV 恢复，或导入你先前导出的本地 JSON 备份。
  • 同步异常：检查网络与登录状态，查看“日志”获取详细错误并定位。
  • 性能建议：大量记录时建议定期清理无用数据，合理设置并行与阈值。
