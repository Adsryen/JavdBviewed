快速开始

欢迎使用 Jav 助手！这是一个功能强大的浏览器扩展，帮助你更好地管理和追踪你的观影记录。

  • 打开控制面板：点击浏览器工具栏中的扩展图标，选择"控制面板"进入主界面。
  • 首次配置：进入"设置"标签页，根据需要配置各项功能：
    • WebDAV 同步 - 实现多设备数据同步
    • 115 网盘 - 一键推送磁力链接下载
    • 功能增强 - 磁链聚合、快速复制等
    • 隐私保护 - 锁屏模式、敏感内容模糊
  • 开始使用：访问 JavDB 网站，扩展会自动在页面上添加标记按钮和增强功能。
  • 查看数据：在控制面板的"番号库"、"演员库"、"新作"标签页中管理你的记录。

数据管理

强大的本地数据库，让你轻松管理所有观影记录。

📚 番号库

  • 搜索与筛选：支持番号、标题、演员名搜索，可按标签、状态、日期筛选。
  • 批量操作：使用 Ctrl/Shift 多选，批量标记状态、添加标签、删除记录。
  • 排序与分组：按添加时间、更新时间、番号排序；支持按状态、标签分组查看。
  • 详情编辑：点击记录可查看详情，编辑标签、备注、评分等信息。
  • 快速标记：使用快捷键 Ctrl+Shift+M 标记"已观看"，Ctrl+Shift+W 标记"想看"。

👤 演员库

  • 收藏同步：自动同步 JavDB 收藏的演员到本地，支持手动刷新。
  • 别名管理：为演员添加自定义别名，方便搜索和识别。
  • 分类整理：创建自定义分类（如"最爱"、"新人"等），拖拽排序。
  • 卡片视图：支持大、中、小三种卡片尺寸，分页浏览。
  • 快速跳转：点击演员卡片可直接跳转到 JavDB 演员页面。

🆕 新作发现

  • 自动订阅：订阅关注的演员，自动发现并展示他们的新作品。
  • 状态联动：新作状态与番号库同步，已标记的作品自动过滤。
  • 过滤选项：按演员、发布日期、状态筛选新作。
  • 清理已读：一键清理已标记或已过期的新作通知。

💾 数据导入导出

  • 本地备份：导出为 JSON 文件，保存到本地磁盘。
  • 数据还原：从 JSON 文件导入，恢复历史数据。
  • WebDAV 云备份：自动或手动上传到云端，多设备同步。
  • 选择性备份：可选择备份内容（设置、记录、演员、日志等）。

📊 统计与报告

  • 首页概览：实时显示"已观看"、"想看"、"想再看"等统计数据。
  • 图表分析：观看趋势图、标签分布图、演员排行等可视化图表。
  • AI 报告：启用 AI 功能后，自动生成个性化观影分析报告。
  • 时间范围：支持查看不同时间段（7天、30天、90天、全部）的统计。

账户信息

管理你的 JavDB 账户信息，用于数据同步。

  • 账户登录：显示并管理你的 JavDB 账户信息（邮箱/用户名/用户类型等）。
  • 安全存储：账户凭据仅在本地存储，仅用于与 JavDB 的数据同步。
  • 登录状态：实时显示登录状态，过期时会提示重新登录。
  • 隐私保护：所有数据仅在本地处理，不会上传到第三方服务器。

数据同步（JavDB ↔ 本地）

与 JavDB 网站双向同步你的观看记录和收藏。

  • 同步全部：一键从 JavDB 获取"已观看 + 想看"的所有记录到本地。
  • 分类同步：可单独拉取"想看"或"已看"的记录，按需更新。
  • 收藏演员：支持同步收藏演员列表，并可单独刷新演员信息。
  • 进度显示：显示"获取列表/拉取详情"两阶段进度，支持随时取消。
  • 增量同步：智能识别新增和更新的记录，避免重复拉取。
  • 冲突处理：本地与云端数据冲突时，可选择保留哪一方的数据。

WebDAV 备份与恢复

通过 WebDAV 协议实现多设备间的数据同步和备份。

  • 自动备份：可设置自动备份间隔（如每天、每周），无需手动操作。
  • 手动备份：随时手动上传当前数据到云端，确保数据安全。
  • 服务兼容：兼容 Nextcloud、Seafile、TeraCloud、Yandex Disk 等主流 WebDAV 服务。
  • 恢复模式：提供三种恢复模式：
    • 快速替换 - 直接用云端数据覆盖本地
    • 智能合并 - 合并本地和云端数据，保留最新
    • 专家模式 - 查看差异，手动选择保留哪些数据
  • 选择性备份：可分别备份"扩展设置"、"观看记录"、"账户信息"、"演员库"、"日志"、"新作"等数据。
  • 版本管理：支持查看备份历史，恢复到指定版本。
  • 加密选项：可选择加密备份数据，保护隐私。

115 网盘集成

与 115 网盘深度集成，实现一键离线下载。

  • 一键推送：在 JavDB 详情页点击按钮，直接将磁力链接推送至 115 离线下载。
  • 验证码处理：自动处理 115 的验证流程，失败可重试。
  • 状态联动：推送成功后可自动标记为"已观看"或其他状态。
  • 配额显示：启用 115-V2 后，在侧边栏实时显示网盘配额（总量/已用/剩余）。
  • 任务管理：查看推送历史和下载任务状态。
  • 批量推送：支持批量选择多个磁力链接，一次性推送。

Emby 增强

在 Emby 媒体服务器中增强显示和跳转功能。

  • 信息联动：在 Emby 页面自动识别番号，显示相关信息。
  • 快速跳转：从 Emby 一键跳转到 JavDB 对应页面。
  • 演员信息：显示演员的收藏状态和相关作品。
  • 标记同步：在 Emby 中标记的观看状态可同步到扩展。

显示与内容过滤

自定义 JavDB 页面的显示效果，过滤不需要的内容。

  • 自动隐藏：访问 JavDB 时自动隐藏"已看"、"已阅"、"VR"等条目，精简列表。
  • 标记样式：自定义标记的颜色、位置、大小，打造个性化界面。
  • 密度控制：调整列表密度（紧凑/标准/宽松），适应不同屏幕。
  • 过滤规则：创建自定义过滤规则，按标签、演员、类型过滤。
  • 显示切换：快速切换显示/隐藏已标记内容。

搜索引擎跳转

配置外部搜索引擎，快速在多个站点搜索番号。

  • 自定义引擎：添加自定义搜索引擎，使用 {{ID}} 作为番号占位符。
  • 预设引擎：内置常用搜索引擎（JavBus、JavLibrary 等）。
  • 图标显示：为每个引擎设置图标，直观识别。
  • 排序管理：拖拽调整搜索引擎的显示顺序。
  • 快速跳转：在番号旁显示搜索按钮，一键跳转到对应站点。
  • 新标签打开：支持在新标签页或当前页打开搜索结果。

功能增强

多种实用功能，提升浏览和管理效率。

  • 磁链聚合：自动聚合站内外磁链资源并去重，一次性查看所有下载源。
  • 快速复制：一键复制番号、标题、磁链等信息，方便整理和分享。
  • 键盘快捷键：支持键盘快捷操作（上一页/下一页、快速标记等）。

隐私保护

保护你的隐私，避免敏感内容被他人看到。

  • 锁屏模式：一键模糊所有敏感区域（封面、标题、演员名等）。
  • 自动隐藏：切换标签页后自动启用隐私模式，回到页面时恢复。
  • 模糊强度：可调整模糊程度（轻度/中度/重度）。
  • 选择性模糊：选择要模糊的内容类型（封面/标题/演员/标签）。
  • 快捷键切换：设置快捷键快速开启/关闭隐私模式。
  • 密码保护：可设置密码，解锁后才能查看内容。

快捷键

使用键盘快捷键，提升操作效率。

  • Ctrl + Shift + M - 快速标记当前条目为"已观看"
  • Ctrl + Shift + W - 快速标记当前条目为"想看"
  • Ctrl + Shift + B - 快速标记当前条目为"想再看"
  • Ctrl + Shift + L - 开启/关闭隐私锁屏模式

  • 多选操作：按住 Ctrl 点击多选，按住 Shift 范围选择
  • 自定义快捷键：在设置中可自定义所有快捷键组合

常见问题

遇到问题？这里有一些常见问题的解决方案。

❓ 数据相关

  • 数据丢失怎么办？可通过 WebDAV 恢复云端备份，或导入之前导出的本地 JSON 备份文件。
  • 如何迁移到新设备？在旧设备导出数据或上传到 WebDAV，在新设备导入或下载即可。
  • 数据占用空间太大？定期清理无用数据、日志，或使用"高级工具"中的缓存清理功能。

🔄 同步相关

  • 同步失败怎么办？检查网络连接和 JavDB 登录状态，查看"日志"获取详细错误信息。
  • WebDAV 连接不上？使用"网络测试"功能检查连接，确认服务器地址、用户名、密码正确。
  • 同步速度慢？调整并发数和批次大小，或选择网络状况较好的时间同步。

⚙️ 功能相关

  • 115 推送失败？检查 115 账号登录状态，确认验证码处理正确，必要时重新登录。
  • AI 功能不可用？检查 AI 设置中的 API Key 是否正确，模型是否可用。
  • 标记按钮不显示？刷新页面，或检查"功能增强"设置是否启用。

💡 性能优化

  • 扩展运行卡顿？关闭不需要的功能，减少同时处理的数据量。
  • 大量记录时建议：定期清理无用数据，合理设置并行与阈值参数。
  • 浏览器占用内存高？定期重启浏览器，清理扩展缓存。
