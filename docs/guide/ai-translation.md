# AI 翻译

AI 翻译功能使用人工智能模型翻译视频标题、简介、标签等内容，帮助你更好地理解日文内容。

## 功能概述

- 支持多种 AI 模型翻译
- 翻译标题、简介、标签、演员名
- 传统翻译和 AI 翻译结合
- 翻译结果缓存
- 自定义翻译提供商

## 支持的内容

### 标题翻译

翻译视频标题：

1. 在详情页点击标题旁的"翻译"按钮
2. 等待翻译完成
3. 显示翻译结果

### 简介翻译

翻译视频简介：

1. 点击简介区域的"翻译"按钮
2. 翻译完整的简介内容
3. 保留原文对照

### 标签翻译

翻译视频标签：

1. 开启"标签翻译"功能
2. 自动翻译所有标签
3. 鼠标悬停显示原文

### 演员名翻译

翻译演员名称：

1. 开启"演员名翻译"功能
2. 显示演员的中文名
3. 保留日文原名

## 翻译提供商

### 传统翻译

使用传统翻译服务：

- 质量稳定

- 翻译质量高

## 配置翻译

### 选择翻译提供商

1. 打开设置面板
2. 进入"AI翻译"选项卡
3. 选择翻译提供商

### 配置 API

如果使用 AI 翻译，需要配置 API：

#### OpenAI 配置

1. 获取 OpenAI API 密钥
   - 访问 https://platform.openai.com
   - 注册账号
   - 创建 API 密钥
2. 在设置中填写：
   - **API 密钥**: 你的 API 密钥
   - **API 地址**: 默认或自定义
   - **模型**: 选择使用的模型

#### Claude 配置

1. 获取 Claude API 密钥
   - 访问 https://console.anthropic.com
   - 注册账号
   - 创建 API 密钥
2. 在设置中填写：
   - **API 密钥**: 你的 API 密钥
   - **模型**: 选择使用的模型

### 翻译质量设置

调整翻译质量：

- **速度优先**: 使用较快的模型
- **平衡**: 推荐设置
- **质量优先**: 使用最好的模型

### 缓存设置

配置翻译缓存：

1. 开启"缓存翻译结果"
2. 设置缓存过期时间
3. 相同内容不会重复翻译

**推荐设置**:
- 缓存过期时间: 30 天
- 最大缓存数量: 1000 条

## 使用翻译

### 手动翻译

在需要时手动触发翻译：

1. 点击"翻译"按钮
2. 等待翻译完成
3. 查看翻译结果

### 自动翻译

自动翻译页面内容：

1. 在设置中开启"自动翻译"
2. 选择要自动翻译的内容
3. 打开页面时自动翻译

**可选内容**:
- 视频标题
- 视频简介
- 视频标签
- 演员名称

### 翻译显示模式

选择翻译结果的显示方式：

#### 追加模式

在原文后追加翻译：

```
原文标题
中文翻译
```

#### 替换模式

用翻译替换原文：

```
中文翻译
```

点击可切换显示原文

#### 对照模式

原文和翻译并排显示：

```
原文标题 | 中文翻译
```

### 翻译目标选择

选择需要翻译的页面元素：

- **列表页**: 标题、标签
- **详情页**: 标题、简介、标签、演员名
- **演员页**: 演员名、简介
- **搜索页**: 搜索结果

## 高级功能

### 自定义提示词

自定义 AI 翻译的提示词：

1. 在设置中找到"翻译提示词"
2. 编辑提示词模板
3. 保存配置

**默认提示词**:
```
请将以下日文翻译成中文，保持原意，使用自然流畅的表达：

{text}
```

### 术语表

添加专业术语翻译：

1. 创建术语表
2. 添加术语对照
3. 翻译时自动应用

**示例**:
```
単体作品 -> 单体作品
中出し -> 中出
```

### 翻译历史

查看翻译历史记录：

1. 打开"翻译历史"
2. 查看所有翻译记录
3. 可以重新使用历史翻译

### 批量翻译

批量翻译多个内容：

1. 选择要翻译的内容
2. 点击"批量翻译"
3. 等待全部完成

## 使用技巧

### 选择合适的提供商

根据需求选择：

- **快速浏览**: 使用 Google 翻译
- **深度阅读**: 使用 AI 翻译
- **成本考虑**: 使用免费服务

### 优化翻译质量

提高翻译质量：

1. 使用更好的 AI 模型
2. 添加术语表
3. 自定义提示词
4. 提供上下文

### 节省成本

如果使用付费 API：

1. 启用翻译缓存
2. 只翻译必要内容
3. 使用较便宜的模型
4. 设置每日限额

### 处理翻译错误

如果翻译不准确：

1. 尝试其他翻译提供商
2. 调整提示词
3. 添加术语表
4. 手动修正

## 故障排查

### 翻译失败

检查：

1. API 密钥是否正确
2. 网络连接是否正常
3. API 额度是否用完
4. 查看错误提示

### 翻译很慢

优化：

1. 使用更快的模型
2. 减少翻译内容长度
3. 启用缓存
4. 检查网络速度

### 翻译质量差

改进：

1. 使用更好的模型
2. 调整提示词
3. 添加术语表
4. 提供更多上下文

### API 费用过高

控制成本：

1. 使用较便宜的模型
2. 启用缓存减少请求
3. 只翻译必要内容
4. 设置每日限额

## 常见问题

### 需要付费吗？

传统翻译服务免费，AI 翻译需要 API 密钥和费用。

### 哪个翻译质量最好？

AI 翻译（如 GPT-4、Claude）质量最好，但需要付费。

### 翻译会保存吗？

会缓存翻译结果，但不会永久保存。

### 可以离线翻译吗？

不可以，所有翻译都需要网络连接。

### 支持其他语言吗？

目前主要支持日文到中文，其他语言支持计划中。

### 翻译数据会上传吗？

使用 AI 翻译时会发送内容到 API 服务器，传统翻译也需要联网。
