# 磁力搜索

磁力搜索功能帮助你从多个来源自动搜索视频的磁力链接，快速找到高质量资源。

## 功能概述

- 多源自动搜索磁力链接
- 支持自定义搜索引擎
- 搜索结果聚合和排序
- 一键复制磁力链接
- 与115网盘推送集成

## 启用磁力搜索

### 1. 打开设置

1. 打开设置面板
2. 进入"页面增强"选项卡
3. 找到"磁力搜索"设置

### 2. 启用功能

1. 开启"启用磁力搜索"开关
2. 配置搜索引擎
3. 设置搜索参数

## 搜索引擎配置

### 内置搜索引擎

扩展内置了多个常用磁力搜索引擎：

- **Sukebei**: 专业的成人内容磁力搜索
- **BTDigg**: DHT 网络搜索
- **BTSOW**: 磁力链接搜索
- **Torrentz2**: 聚合搜索引擎

### 启用搜索引擎

1. 在"磁力搜索"设置中
2. 勾选要使用的搜索引擎
3. 可以同时启用多个引擎

### 添加自定义引擎

添加其他磁力搜索网站：

1. 点击"添加自定义引擎"
2. 填写引擎信息：
   - **名称**: 搜索引擎名称
   - **URL 模板**: 搜索 URL（使用 `{query}` 作为关键词占位符）
   - **图标**: 搜索引擎图标 URL（可选）
3. 保存配置

**URL 模板示例**:
```
https://example.com/search?q={query}
```

### 管理搜索引擎

- **排序**: 拖动调整搜索引擎顺序
- **编辑**: 修改引擎配置
- **删除**: 删除自定义引擎
- **启用/禁用**: 切换引擎状态

## 使用磁力搜索

### 自动搜索

在视频详情页自动搜索：

1. 打开视频详情页
2. 扩展会自动触发磁力搜索
3. 搜索结果显示在页面中

⚠️ **待优化**: 已看状态默认不加载，改为手动触发

### 手动搜索

手动触发磁力搜索：

1. 在详情页找到磁力搜索区域
2. 点击"搜索磁力"按钮
3. 等待搜索完成

### 浮动按钮

使用浮动搜索按钮：

1. 在详情页右侧显示浮动按钮
2. 点击按钮触发搜索
3. 可以拖动按钮位置

## 搜索结果

### 结果展示

搜索结果包含以下信息：

- **文件名**: 磁力链接的文件名
- **文件大小**: 文件大小（GB/MB）
- **发布时间**: 资源发布时间
- **做种数**: 当前做种人数
- **下载数**: 下载次数
- **来源**: 搜索引擎来源
- **磁力链接**: 完整的磁力链接

### 结果排序

可以按以下方式排序：

- **文件大小**: 从大到小（通常大文件质量更好）
- **发布时间**: 从新到旧
- **做种数**: 从多到少（做种多下载快）
- **相关度**: 按相关性排序

### 结果筛选

筛选搜索结果：

- **最小文件大小**: 过滤小文件
- **最大文件大小**: 过滤超大文件
- **关键词**: 按文件名关键词筛选
- **来源**: 按搜索引擎筛选

## 操作磁力链接

### 复制磁力链接

1. 找到想要的磁力链接
2. 点击"复制"按钮
3. 磁力链接已复制到剪贴板

### 推送到115网盘

如果启用了115网盘功能：

1. 点击磁力链接旁的"推送115"按钮
2. 等待推送完成
3. 推送成功后自动标记（如果启用）

### 在新标签页打开

1. 点击磁力链接
2. 在新标签页打开搜索引擎页面
3. 查看更多详细信息

## 搜索优化

### 并发控制

控制同时搜索的引擎数量：

1. 在"高级设置"中
2. 设置"磁力搜索并发数"
3. 推荐值：2-3个

**说明**:
- 并发数越大，搜索越快
- 但可能导致请求被限制
- 根据网络情况调整

### 超时设置

设置搜索超时时间：

1. 在"高级设置"中
2. 设置"搜索超时时间"（秒）
3. 推荐值：30-60秒

### 结果数量限制

限制每个引擎返回的结果数：

1. 设置"每个引擎最大结果数"
2. 推荐值：10-20个
3. 减少结果数量可以加快加载

### 缓存搜索结果

启用搜索结果缓存：

1. 开启"缓存搜索结果"
2. 设置缓存过期时间
3. 相同视频不会重复搜索

## 高级功能

### 搜索关键词优化

扩展会自动优化搜索关键词：

- 使用番号作为主要关键词
- 添加演员名称提高准确度
- 过滤特殊字符
- 支持多种番号格式

### 结果去重

自动去除重复的搜索结果：

- 根据磁力链接 hash 去重
- 根据文件名相似度去重
- 保留质量最好的结果

### 智能推荐

根据以下因素推荐最佳磁力：

- 文件大小（越大通常质量越好）
- 做种数（越多下载越快）
- 发布时间（越新越好）
- 文件名完整度

## 使用技巧

### 选择最佳磁力

判断磁力链接质量：

1. **文件大小**: 
   - 1080p: 通常 3-8GB
   - 4K: 通常 8-20GB
   - 太小可能质量差

2. **做种数**:
   - 做种数越多，下载速度越快
   - 至少选择有做种的资源

3. **发布时间**:
   - 越新的资源越好
   - 但新资源做种可能较少

4. **文件名**:
   - 包含清晰度信息（1080p、4K）
   - 包含字幕信息（中文字幕）
   - 文件名完整规范

### 快速搜索工作流

1. 打开视频详情页
2. 等待自动搜索完成
3. 按文件大小排序
4. 选择合适的磁力链接
5. 推送到115网盘

### 批量搜索

如果需要搜索多个视频：

1. 在多个标签页打开视频
2. 每个页面会自动搜索
3. 依次查看搜索结果
4. 批量推送到115

### 处理搜索失败

如果搜索失败或无结果：

1. 检查网络连接
2. 尝试其他搜索引擎
3. 手动在搜索引擎网站搜索
4. 调整搜索关键词

## 故障排查

### 搜索无结果

可能的原因：

1. **视频太新**: 还没有磁力资源
2. **番号错误**: 检查番号是否正确
3. **搜索引擎问题**: 尝试其他引擎
4. **网络问题**: 检查网络连接

### 搜索很慢

优化方法：

1. 减少启用的搜索引擎数量
2. 降低并发搜索数
3. 减少结果数量限制
4. 启用搜索结果缓存

### 搜索结果不准确

改进方法：

1. 使用更精确的番号
2. 添加演员名称
3. 调整搜索引擎优先级
4. 使用结果筛选功能

### 无法复制磁力链接

检查：

1. 浏览器剪贴板权限
2. 是否有安全软件阻止
3. 尝试手动选择复制

## 常见问题

### 为什么有些视频搜索不到？

可能原因：
- 视频太新，还没有资源
- 视频太冷门，没人分享
- 搜索引擎没有收录

### 可以搜索其他网站的视频吗？

目前只支持 JavDB，其他网站支持计划中。

### 搜索结果可以保存吗？

搜索结果会缓存一段时间，但不会永久保存。

### 支持 BT 种子下载吗？

目前只支持磁力链接，BT 种子功能计划中。
