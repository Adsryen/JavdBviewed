# 页面增强

页面增强功能优化 JavDB 的浏览体验，提供更多便捷功能和视觉优化。

## 列表页增强

### 点击增强

优化列表项的点击体验：

1. 在设置中开启"点击增强"
2. 点击视频卡片任意位置都可以打开
3. 无需精确点击标题

### 视频预览

鼠标悬停显示视频预览：

1. 开启"视频预览"功能
2. 鼠标悬停在视频卡片上
3. 自动播放预览视频

#### 预览来源选择

选择视频预览的优先来源：

- **auto**: 自动选择最佳来源
- **javdb**: 使用 JavDB 的预览
- **javspyl**: 使用 javspyl 的预览
- **avpreview**: 使用 avpreview 的预览
- **vbgfl**: 使用 vbgfl 的预览

⚠️ **待调整**: 预览来源的选择逻辑需要优化

#### 预览设置

- **预览延迟**: 设置鼠标悬停多久后开始预览
- **预览音量**: 设置预览视频音量
- **预览质量**: 选择预览视频质量

### 右键后台打开

右键点击在后台标签页打开：

1. 开启"右键后台打开"功能
2. 右键点击视频卡片
3. 在后台新标签页打开，不切换标签

⚠️ **待支持**: 影片页的"你可能也喜歡"区域

### 演员水印

在封面上显示演员状态水印：

1. 开启"演员水印"功能
2. 订阅演员显示蓝色铃铛图标
3. 黑名单演员显示红色禁止图标

### 滚动分页

⚠️ **实验性功能**: 自动加载下一页内容

1. 开启"滚动分页"功能
2. 滚动到页面底部时自动加载下一页
3. 实现无限滚动效果

**注意**: 此功能可能不稳定，谨慎使用

## 内容过滤

### 关键词过滤

基于关键词过滤内容：

⚠️ **待优化**: 
- 增加发行日期字段过滤
- Toast输出过滤统计

#### 添加过滤规则

1. 在设置中找到"内容过滤"
2. 点击"添加规则"
3. 填写规则信息：
   - **规则名称**: 便于管理
   - **关键词**: 要过滤的关键词
   - **匹配范围**: 标题/演员/标签/全部
   - **动作**: 隐藏/高亮/标记

#### 规则示例

```
规则名称: 过滤VR
关键词: VR, 虚拟现实
匹配范围: 标签
动作: 隐藏
```

### 正则表达式

支持正则表达式过滤：

⚠️ **待优化**: 增加动态检测与提示

#### 使用正则

1. 在过滤规则中选择"正则表达式"
2. 输入正则表达式
3. 测试并保存

#### 正则示例

```javascript
// 过滤番号包含数字的
/\d{3,}/

// 过滤特定系列
/^(SSIS|SSNI|STARS)-/

// 过滤特定演员
/(演员A|演员B|演员C)/
```

### 过滤规则管理

管理多个过滤规则：

- **启用/禁用**: 切换规则状态
- **编辑**: 修改规则内容
- **删除**: 删除规则
- **排序**: 调整规则优先级
- **导入/导出**: 分享规则配置

### 过滤统计

显示过滤统计信息：

- **总规则数**: 所有过滤规则数量
- **启用规则数**: 当前启用的规则
- **过滤数量**: 被过滤的视频数量
- **过滤率**: 过滤比例

## 常见问题

### 预览视频不播放？

检查：
1. 网络连接
2. 预览来源设置
3. 浏览器是否阻止自动播放

### 过滤不生效？

检查：
1. 规则是否启用
2. 关键词是否正确
3. 匹配范围是否合适
4. 刷新页面

### 页面样式错乱？

解决：
1. 关闭自定义样式
2. 清除浏览器缓存
3. 重新加载扩展
4. 检查CSS语法
