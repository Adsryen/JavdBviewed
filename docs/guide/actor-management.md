# 演员管理

演员管理功能帮助你收藏喜欢的演员、订阅新作品、屏蔽不感兴趣的演员，并智能过滤列表内容。

## 演员库概述

演员库是管理所有演员信息的中心，支持三种分类：

- **收藏演员**: 你喜欢的演员
- **订阅演员**: 自动监控新作品的演员
- **黑名单演员**: 不感兴趣的演员，会自动隐藏

## 查看演员库

1. 打开设置面板
2. 选择"演员库"选项卡
3. 查看所有演员列表

### 演员信息

每个演员记录包含：

- **姓名**: 演员的主要名称
- **别名**: 其他常用名称
- **头像**: 演员照片
- **性别**: 男性/女性
- **分类**: 有码/无码/欧美
- **作品数量**: 已知作品总数
- **标签**: 自定义标签
- **备注**: 详细信息（年龄、身高、罩杯等）

## 添加演员

### 方式一：从视频详情页添加

1. 打开视频详情页
2. 找到演员名称
3. 点击演员名称旁的操作按钮
4. 选择分类：
   - 收藏
   - 订阅
   - 黑名单

### 方式二：从演员页添加

1. 访问演员的个人页面
2. 点击页面上的"添加到演员库"按钮
3. 选择分类

### 方式三：从演员库添加

1. 打开演员库
2. 点击"添加演员"按钮
3. 搜索演员名称
4. 选择演员并确认

### 方式四：从 JavDB 同步

1. 在演员库页面点击"同步演员"
2. 扩展会自动从 JavDB 获取你收藏的演员
3. 同步完成后会显示新增数量

## 演员分类

### 收藏演员

**用途**: 标记你喜欢的演员

**功能**:
- 在列表页优先显示收藏演员的作品
- 可以设置"仅显示收藏演员"过滤
- 在演员页面显示特殊标记

**添加方式**:
1. 点击演员名称旁的"❤️"按钮
2. 或在演员库中标记为"收藏"

### 订阅演员

**用途**: 自动监控演员的新作品

**功能**:
- 自动检测订阅演员的新作品
- 有新作品时显示通知
- 在"新作品"页面查看
- 可以设置"订阅视为收藏"

**添加方式**:
1. 点击演员名称旁的"🔔"按钮
2. 或在演员库中标记为"订阅"

**订阅设置**:
1. 打开设置面板
2. 进入"新作品"选项卡
3. 配置检测频率和通知方式

### 黑名单演员

**用途**: 屏蔽不感兴趣的演员

**功能**:
- 列表页自动隐藏黑名单演员的作品
- 搜索结果中不显示
- 新作品检测中排除

**添加方式**:
1. 点击演员名称旁的"🚫"按钮
2. 或在演员库中标记为"黑名单"

**注意**: 
- 黑名单演员的作品会完全隐藏
- 可以在演员库中查看和管理黑名单
- 演员页面不受影响（避免误操作）

## 演员标签

### 添加标签

为演员添加自定义标签：

1. 在演员库中点击演员
2. 点击"添加标签"
3. 输入标签名称（如：新人、退役、最爱等）
4. 确认添加

### 标签筛选

使用标签快速筛选演员：

1. 在演员库顶部的标签栏
2. 点击标签名称
3. 只显示有该标签的演员

### 标签管理

1. 点击"管理标签"按钮
2. 可以重命名、删除标签
3. 批量为演员添加/移除标签

## 演员备注

### 添加备注

为演员添加详细信息：

1. 点击演员条目
2. 点击"编辑备注"按钮
3. 填写信息：
   - **年龄**: 出生年份或当前年龄
   - **身高**: 厘米
   - **罩杯**: A-Z
   - **三围**: B-W-H
   - **其他**: 自定义备注

### 备注显示

- 在演员库列表中显示简要信息
- 鼠标悬停显示完整备注
- 在演员页面显示详细信息

### 备注缓存

扩展会自动缓存演员备注：

- 减少重复查询
- 提升加载速度
- 可设置缓存过期时间（TTL）

**配置缓存**:
1. 打开设置面板
2. 进入"高级设置"选项卡
3. 设置"演员备注缓存时长"（小时）

## 演员头像

### 自动获取

扩展会自动获取演员头像：

- 从 JavDB 获取
- 从其他数据源聚合
- 自动缓存到本地

### 头像缓存

启用头像缓存可以：

- 加快页面加载速度
- 减少网络请求
- 离线查看

**启用缓存**:
1. 打开设置面板
2. 进入"高级设置"选项卡
3. 开启"演员头像缓存"

### 更新头像

如果头像显示不正确：

1. 在演员库中找到该演员
2. 点击"更新头像"按钮
3. 扩展会重新获取头像

## 演员过滤

### 列表页过滤

在视频列表页根据演员库过滤作品：

#### 隐藏黑名单演员

1. 打开设置面板
2. 进入"页面增强"选项卡
3. 开启"隐藏黑名单演员的作品"
4. 列表页会自动隐藏

#### 仅显示收藏演员

1. 在"页面增强"设置中
2. 开启"仅显示收藏演员的作品"
3. 列表页只显示收藏演员的作品

#### 订阅视为收藏

1. 在"演员管理"设置中
2. 开启"订阅演员视为收藏演员"
3. 订阅的演员也会被当作收藏演员过滤

**注意**: 演员页面不会应用过滤，避免因标题匹配误差影响数据显示

### 搜索过滤

在演员库中搜索：

1. 在搜索框输入演员名称
2. 支持模糊搜索
3. 搜索别名也能找到

### 分类过滤

按分类筛选演员：

- **全部**: 显示所有演员
- **收藏**: 只显示收藏的演员
- **订阅**: 只显示订阅的演员
- **黑名单**: 只显示黑名单演员

## 演员页面增强

### 自动应用标签

在演员个人页面自动应用标签过滤：

1. 打开设置面板
2. 进入"演员管理"选项卡
3. 设置"默认标签过滤"
4. 访问演员页面时自动应用

⚠️ **已知问题**: 加载上次应用的标签功能可能未生效

### 默认排序

设置演员页面的默认排序方式：

- **发布时间**: 最新作品在前
- **评分**: 高评分作品在前
- **热度**: 热门作品在前

**配置**:
1. 在"演员管理"设置中
2. 选择"默认排序方式"

### 影片分段显示

按时间段分组显示演员作品：

1. 开启"影片分段显示"
2. 设置"月份阈值"（如：6个月）
3. 作品会按时间段分组显示

**效果**:
```
2024年
  ├─ 1-6月 (10部)
  └─ 7-12月 (8部)

2023年
  ├─ 1-6月 (12部)
  └─ 7-12月 (15部)
```

### 保存过滤器

保存当前演员页面的过滤设置：

1. 在演员页面应用标签过滤
2. 点击"保存过滤器"按钮
3. 下次访问该演员页面时自动应用

### 快速筛选

在演员页面快速切换标签过滤：

1. 页面顶部显示标签快捷按钮
2. 点击标签快速筛选
3. 支持多标签组合

## 演员同步

### 从 JavDB 同步

自动同步 JavDB 的收藏演员：

1. 点击"同步演员"按钮
2. 扩展会获取你在 JavDB 收藏的演员
3. 自动添加到演员库

### 自动同步

设置定期自动同步：

1. 打开设置面板
2. 进入"演员管理"选项卡
3. 开启"自动同步演员"
4. 设置同步间隔（小时）

**推荐间隔**:
- **6小时**: 频繁使用
- **12小时**: 推荐设置
- **24小时**: 偶尔使用

### 同步详情

同步时会更新：

- 演员基本信息
- 头像
- 作品数量
- 别名

### 同步冲突

如果本地和 JavDB 的数据不一致：

- 保留本地的分类（收藏/订阅/黑名单）
- 更新演员的基本信息
- 合并别名列表

## 演员 Wiki 链接

### 自动生成链接

扩展会自动生成演员的 Wiki 链接：

- **Wikipedia**: 演员的维基百科页面
- **xslist**: 演员的详细资料页面

### 访问 Wiki

1. 在演员库中点击演员
2. 点击"查看 Wiki"按钮
3. 在新标签页打开 Wiki 页面

## 批量操作

### 批量添加标签

1. 在演员库中勾选多个演员
2. 点击"批量操作" → "添加标签"
3. 选择或创建标签
4. 确认操作

### 批量修改分类

1. 勾选多个演员
2. 点击"批量操作" → "修改分类"
3. 选择新分类（收藏/订阅/黑名单）
4. 确认操作

### 批量删除

1. 勾选要删除的演员
2. 点击"批量操作" → "删除"
3. 确认删除

⚠️ **注意**: 删除演员不会删除相关的视频记录

## 使用技巧

### 建立演员体系

1. **收藏**: 标记喜欢的演员
2. **订阅**: 关注活跃演员的新作品
3. **黑名单**: 屏蔽不感兴趣的演员
4. **标签**: 按个人喜好分类（新人、退役、最爱等）

### 配合过滤使用

1. 启用"仅显示收藏演员"
2. 列表页只显示收藏演员的作品
3. 减少干扰，提高浏览效率

### 新作品监控

1. 订阅喜欢的演员
2. 启用新作品自动检测
3. 有新作品时收到通知
4. 在"新作品"页面查看

### 演员研究

1. 添加演员备注记录详细信息
2. 使用标签分类管理
3. 查看 Wiki 了解更多信息
4. 在演员页面查看所有作品

## 常见问题

### 演员头像不显示？

1. 检查网络连接
2. 清除头像缓存
3. 点击"更新头像"重新获取

### 同步演员失败？

1. 确认已登录 JavDB
2. 检查网络连接
3. 查看浏览器控制台错误信息

### 过滤不生效？

1. 确认已启用相关设置
2. 刷新页面
3. 检查演员是否在正确的分类中

### 黑名单演员还是显示？

1. 确认已开启"隐藏黑名单演员"
2. 演员页面不会隐藏（设计如此）
3. 检查演员是否真的在黑名单中
