# 视频标记功能

视频标记是 Jav 助手的核心功能，帮助你记录和管理观看状态。

## 标记状态说明

### 三种标记状态

1. **已观看** (✓ 绿色)
   - 表示你已经完整观看过这个视频
   - 手动标记
   - 优先级最高（仅次于"想看"）

2. **想看** (★ 黄色)
   - 表示你对这个视频感兴趣，计划观看
   - 手动标记
   - 优先级最高

3. **已浏览** (灰色)
   - 表示你访问过视频详情页
   - 自动标记（可在设置中关闭）
   - 优先级最低

### 优先级显示规则

当一个视频有多个状态时，按以下优先级显示：

```
想看 > 已观看 > 已浏览
```

例如：如果你标记了"想看"，即使视频也被标记为"已浏览"，列表中只会显示"想看"标记。

## 标记方式

### 方式一：详情页快速按钮

1. 打开视频详情页
2. 页面顶部会显示快捷操作按钮
3. 点击相应按钮即可标记

**优点**: 操作直观，适合仔细查看后标记

### 方式二：列表页快捷按钮

1. 在视频列表页浏览
2. 鼠标悬停在视频卡片上
3. 点击出现的标记按钮

**优点**: 快速批量标记，无需打开详情页

### 方式三：键盘快捷键

在视频详情页使用快捷键：

- `V` 键：标记/取消"已观看"
- `W` 键：标记/取消"想看"

**优点**: 最快速的标记方式，适合键盘操作

### 方式四：右键菜单

1. 右键点击视频卡片
2. 在弹出菜单中选择标记选项

**优点**: 与浏览器操作习惯一致

## 视觉反馈

### Favicon 状态指示

浏览器标签页的图标会根据视频状态变化：

- 🟢 绿色图标：已观看
- 🔵 蓝色图标：想看
- 🟡 黄色图标：已浏览
- ⚪ 原始图标：未标记

### 页面标题前缀

详情页的标题会添加状态标识：

- `[已观看]` - 已观看的视频
- `[我想看]` - 想看的视频
- `[已浏览]` - 已浏览的视频

### 列表标记样式

列表页的视频卡片会显示彩色标记：

- 绿色勾选：已观看
- 黄色星标：想看
- 灰色标记：已浏览

## 高级功能

### 收藏和评分

在视频详情页，标题下方会显示收藏和评分功能：

#### 收藏功能

- 点击"收藏"按钮将视频添加到收藏列表
- 收藏的视频可以在番号库中筛选查看
- 收藏状态独立于观看状态

#### 评分功能

- 点击星星为视频评分（0.5 - 5.0 星）
- 支持半星评分（点击星星左侧为半星，右侧为整星）
- 评分会保存在番号库中
- 可以随时修改评分

### 批量操作

在设置面板的"番号库"中，可以进行批量操作：

1. 勾选多个视频
2. 点击"批量操作"按钮
3. 选择操作类型：
   - 批量标记为已观看
   - 批量标记为想看
   - 批量删除
   - 批量修改清单

### 清单管理

为视频分配到不同的清单：

1. 在详情页或番号库中点击"清单"按钮
2. 选择或创建清单
3. 一个视频可以属于多个清单
4. 支持同步 JavDB 的清单列表

## 自动标记

### 自动标记已浏览

默认情况下，访问视频详情页会自动标记为"已浏览"。

**关闭自动标记**：
1. 打开设置面板
2. 进入"基本设置"选项卡
3. 关闭"自动标记已浏览"开关

### 115推送自动标记

当你使用 115网盘推送功能时，可以设置推送成功后自动标记为"已观看"。

**启用自动标记**：
1. 打开设置面板
2. 进入"115网盘"选项卡
3. 开启"推送成功后自动标记为已观看"

### 想看按钮同步

点击 JavDB 原生的"想看"按钮时，扩展会自动同步状态到番号库。

## 状态同步

### 实时同步

标记状态会实时同步到所有打开的页面：

- 在详情页标记后，列表页会立即更新
- 在列表页标记后，详情页会立即更新
- 多个标签页之间自动同步

### 跨设备同步

配置 WebDAV 后，标记状态可以在多个设备间同步：

1. 参考 [云端同步教程](https://javd-bviewed-docs.vercel.app/guide/webdav-sync) 配置 WebDAV
2. 启用自动同步
3. 所有设备的标记状态会自动保持一致

## 使用技巧

### 快速标记工作流

1. **浏览模式**: 在列表页快速浏览，看到感兴趣的点击"想看"
2. **观看模式**: 从"想看"列表中选择视频观看
3. **标记模式**: 观看后在详情页标记为"已观看"

### 利用优先级

- 先标记"想看"建立观看列表
- 观看后改为"已观看"
- "已浏览"自动记录浏览历史

### 配合过滤使用

在设置中启用"隐藏已观看"功能：

1. 打开设置面板
2. 进入"页面增强"选项卡
3. 开启"隐藏已观看的影片"
4. 列表页会自动隐藏已观看的视频

## 常见问题

### 标记后没有显示？

1. 刷新页面
2. 检查是否有其他优先级更高的状态
3. 查看浏览器控制台是否有错误

### 标记丢失了？

1. 检查是否清除了浏览器数据
2. 从备份或 WebDAV 恢复数据
3. 查看番号库确认数据是否存在

### 快捷键不生效？

1. 确保焦点在页面上（不在输入框中）
2. 检查是否与其他扩展冲突
3. 尝试刷新页面
