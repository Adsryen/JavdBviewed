# 数据管理

Jav 助手提供完善的数据管理功能，包括番号库、演员库、数据导入导出和统计分析。

## 番号库管理

番号库记录了所有你标记过的视频信息。

### 查看番号库

1. 点击扩展图标打开弹窗
2. 点击"设置"进入设置面板
3. 选择"番号库"选项卡

### 筛选功能

#### 按状态筛选

- **全部**: 显示所有记录
- **已观看**: 只显示标记为"已观看"的视频
- **想看**: 只显示标记为"想看"的视频
- **已浏览**: 只显示标记为"已浏览"的视频

#### 按清单筛选

1. 点击"清单"下拉菜单
2. 选择要查看的清单
3. 支持查看多个清单

#### 搜索功能

在搜索框中输入：

- 番号（如：SSIS-001）
- 视频标题关键词
- 演员名称

### 批量操作

#### 批量选择

1. 勾选要操作的视频
2. 或点击"全选"按钮选择当前页所有视频

#### 批量修改状态

1. 选择多个视频
2. 点击"批量操作"按钮
3. 选择"修改状态"
4. 选择新状态（已观看/想看/已浏览）

#### 批量删除

1. 选择要删除的视频
2. 点击"批量操作" → "删除"
3. 确认删除操作

⚠️ **注意**: 删除操作不可恢复，建议先导出备份

#### 批量修改清单

1. 选择多个视频
2. 点击"批量操作" → "修改清单"
3. 选择或创建清单
4. 确认操作

### 清单管理

#### 同步 JavDB 清单

1. 在番号库页面点击"同步清单"按钮
2. 扩展会自动从 JavDB 获取你的清单列表
3. 同步完成后可以在筛选中使用

#### 创建自定义清单

1. 在番号库中选择视频
2. 点击"清单"按钮
3. 输入新清单名称
4. 确认创建

#### 清单归属

- 一个视频可以属于多个清单
- 在视频详情中可以查看所属清单
- 支持从清单中移除视频

### 详细信息

点击视频条目可以查看详细信息：

- **番号**: 视频唯一标识
- **标题**: 视频标题
- **状态**: 当前标记状态
- **清单**: 所属清单列表
- **收藏状态**: 是否收藏
- **用户评分**: 你的评分（0-5星）
- **标记时间**: 首次标记时间
- **更新时间**: 最后修改时间
- **访问次数**: 访问详情页的次数

## 演员库管理

演员库用于管理你关注的演员信息。

### 查看演员库

1. 打开设置面板
2. 选择"演员库"选项卡

### 演员分类

#### 收藏演员

- 你喜欢的演员
- 可以设置"仅显示收藏演员"过滤列表

#### 订阅演员

- 自动监控新作品的演员
- 有新作品时会收到通知

#### 黑名单演员

- 不感兴趣的演员
- 列表页会自动隐藏这些演员的作品

### 添加演员

#### 方式一：从视频详情页添加

1. 打开视频详情页
2. 点击演员名称旁的"+"按钮
3. 选择分类（收藏/订阅/黑名单）

#### 方式二：从演员库添加

1. 打开演员库
2. 点击"添加演员"按钮
3. 搜索或输入演员名称
4. 选择分类并确认

### 演员信息

每个演员记录包含：

- **姓名**: 演员名称
- **别名**: 其他常用名称
- **性别**: 男性/女性
- **分类**: 有码/无码/欧美
- **头像**: 演员照片
- **作品数量**: 已知作品总数
- **备注**: 自定义备注信息

### 演员备注

为演员添加详细信息：

1. 点击演员条目
2. 点击"编辑备注"
3. 填写信息：
   - 年龄
   - 身高
   - 罩杯
   - 其他备注
4. 保存

### 演员同步

#### 从 JavDB 同步

1. 点击"同步演员"按钮
2. 扩展会从 JavDB 获取你收藏的演员
3. 自动更新演员信息

#### 自动同步设置

1. 打开设置面板
2. 进入"演员管理"选项卡
3. 设置自动同步间隔（小时）
4. 启用"自动同步演员"

## 数据导入导出

### 导出数据

#### 完整导出

1. 打开设置面板
2. 进入"数据管理"选项卡
3. 点击"导出数据"按钮
4. 选择导出格式：
   - **JSON**: 完整数据，可用于恢复
   - **CSV**: 表格格式，可用于分析

#### 选择性导出

1. 点击"高级导出"
2. 选择要导出的数据类型：
   - 番号库
   - 演员库
   - 新作品记录
   - 设置配置
3. 点击"导出选中项"

#### 筛选条件导出

在番号库或演员库中：

1. 应用筛选条件（状态、清单、搜索等）
2. 点击"导出当前筛选结果"
3. 只导出符合条件的数据

### 导入数据

#### 从备份恢复

1. 打开设置面板
2. 进入"数据管理"选项卡
3. 点击"导入数据"按钮
4. 选择之前导出的 JSON 文件
5. 选择导入模式：
   - **合并**: 保留现有数据，添加新数据
   - **覆盖**: 清空现有数据，使用导入数据
6. 确认导入

#### 导入进度

导入大量数据时会显示进度：

- 已处理条目数
- 总条目数
- 预计剩余时间

### 数据合并策略

当选择"合并"模式时：

- **番号库**: 相同番号的记录会合并，保留最新状态
- **演员库**: 相同演员会合并，保留所有分类
- **设置**: 不会覆盖现有设置

## 统计信息

### 查看统计

在设置面板首页可以看到：

#### 基本统计

- **总观看数**: 标记为"已观看"的视频数量
- **想看数**: 标记为"想看"的视频数量
- **已浏览数**: 标记为"已浏览"的视频数量
- **收藏演员数**: 收藏的演员数量
- **订阅演员数**: 订阅的演员数量

#### 时间统计

- **本月观看**: 本月标记为"已观看"的数量
- **本周观看**: 本周标记为"已观看"的数量
- **今日观看**: 今天标记为"已观看"的数量

#### 存储统计

- **数据占用**: 扩展数据占用的存储空间
- **缓存占用**: 图片和其他缓存占用的空间
- **总占用**: 总存储空间占用

### 数据可视化

首页提供多种图表：

- **观看趋势图**: 显示每日/每周/每月观看数量变化
- **演员分布图**: 显示不同演员的作品观看数量
- **标签分布图**: 显示不同标签的作品数量
- **时间分布图**: 显示不同时间段的观看习惯

## 数据备份建议

### 定期备份

建议每月至少备份一次数据：

1. 导出完整 JSON 数据
2. 保存到安全位置（云盘、本地硬盘）
3. 保留多个版本的备份

### 备份提醒

扩展会在以下情况提醒你备份：

- 距离上次备份超过 30 天
- 数据量增长超过 1000 条
- 即将进行危险操作（如批量删除）

### 云端同步

配置 WebDAV 后，数据会自动同步到云端：

- 参考 [云端同步教程](/guide/webdav-sync)
- 自动备份，无需手动操作
- 支持多设备同步

## 数据清理

### 清理缓存

1. 打开设置面板
2. 进入"数据管理"选项卡
3. 点击"清理缓存"
4. 选择要清理的缓存类型：
   - 图片缓存
   - 演员头像缓存
   - 翻译缓存
5. 确认清理

### 清理旧数据

清理长时间未访问的数据：

1. 点击"清理旧数据"
2. 设置时间阈值（如：6个月未访问）
3. 预览将被清理的数据
4. 确认清理

⚠️ **注意**: 清理前建议先导出备份

## 常见问题

### 数据丢失了怎么办？

1. 检查是否有备份文件
2. 如果配置了 WebDAV，从云端恢复
3. 查看浏览器是否有自动备份

### 导入数据失败？

1. 检查文件格式是否正确（必须是 JSON）
2. 确认文件没有损坏
3. 查看浏览器控制台的错误信息

### 数据占用空间太大？

1. 清理图片缓存
2. 清理长时间未访问的数据
3. 导出后删除部分旧数据
