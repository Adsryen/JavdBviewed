# 显示设置

显示设置对应仪表盘中的 `设置 → 显示设置`，主要控制列表展示、过滤行为和页面视觉呈现。

## 适用内容

- 影片卡片显示方式
- 列表列数、容器宽度等展示参数
- 过滤条件和默认显示行为
- 与浏览体验直接相关的基础显示配置

## 推荐阅读

- [界面介绍](/guide/interface)
- [视频标记](/guide/video-marking)
- [数据管理](/guide/data-management)
- [页面增强](/guide/page-enhancement)

## 使用建议

1. 先在 `显示设置` 中调整基础显示参数
2. 再到 [页面增强](/guide/page-enhancement) 开启进阶体验功能
3. 如果需要同步这些设置，再查看 [WebDAV 同步](/guide/webdav-sync)
