# 同步设置

同步设置对应仪表盘中的 `设置 → 同步设置`，用于配置 JavDB 数据抓取、同步范围和同步策略。

## 进入位置

- 仪表盘 → `设置`
- 选择 `同步设置`

## 主要用途

- 配置需要同步的数据类型
- 调整同步频率和策略
- 管理同步范围与覆盖规则
- 配合 [WebDAV 同步](/guide/webdav-sync) 完成云端备份和恢复
