Emby/Jellyfin 增强设置

配置 Emby/Jellyfin 等媒体服务器的番号识别和跳转功能，自动将页面中的番号转换为可点击的链接。

影音增强功能可能还存在问题，如遇到任何异常请在 GitHub Issues 反馈。

基本设置

启用 Emby/Jellyfin 增强功能

启用后，扩展将在匹配的网站上自动识别番号并转换为可点击的链接。

网址匹配

配置需要启用番号识别功能的网址模式。支持通配符 * 匹配任意字符。

链接行为

选择点击番号链接后的跳转行为。推荐使用搜索模式以确保能找到相关内容。

快捷按钮

在右侧显示悬浮按钮，快速按页面内容或选中文本进行番号搜索/直达。

在右侧显示悬浮按钮，快速按页面内容或选中文本进行演员搜索。

媒体库入库状态

在 JavDB 列表页和详情页显示本地媒体服务器是否已入库。

后台自动同步会按此间隔使用本地保存的 API Key 请求已启用服务器的影片列表，并把入库索引保存到扩展本地；点击“立即同步媒体库”会马上执行一次。

媒体服务器

配置 Emby/Jellyfin 服务器地址和 API Key。

使用说明

  • 网址模式: 使用通配符 * 匹配任意字符，如 *.emby.com/* 匹配所有emby.com的子域名
  • 番号识别: 自动识别常见的番号格式，如 ABC-123、FC2-PPV-123456 等
  • 链接跳转: 所有链接都会在新标签页中打开，不影响当前页面
  • 兼容性: 支持 Emby、Jellyfin 等主流媒体服务器
