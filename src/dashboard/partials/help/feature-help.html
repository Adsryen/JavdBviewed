快速开始

欢迎使用 Jav 助手！这是一个功能强大的浏览器扩展，帮助你更好地管理和追踪你的观影记录。

  • 打开控制面板：点击浏览器工具栏中的扩展图标，选择"控制面板"进入主界面。也可以通过快捷键 Alt+J 快速打开。
  • 首次配置：进入"设置"标签页，根据需要配置各项功能：
    • WebDAV 同步 - 实现多设备数据同步，支持 Nextcloud、Seafile 等服务
    • 115 网盘 - 一键推送磁力链接下载，支持自动验证码处理
    • 功能增强 - 磁链聚合、快速复功能
    • 隐私保护 - 锁屏模式、敏感内容模糊、密码保护
    • AI 功能 - 个性化观影报告、智能推荐（需配置 API Key）
  • 开始使用：访问 JavDB 网站，扩展会自动在页面上添加标记按钮和增强功能。首次使用建议先同步 JavDB 的观看记录。
  • 查看数据：在控制面板的"番号库"、"演员库"、"新作"标签页中管理你的记录。支持搜索、筛选、排序等多种操作。
  • 数据备份：建议定期导出数据或配置 WebDAV 自动备份，避免数据丢失。

💡 新手提示

  • 首次同步：建议先在"设置 → 数据同步"中点击"同步全部"，将 JavDB 的历史记录导入到本地。
  • 演员订阅：在"演员库"中收藏喜欢的演员，系统会自动追踪他们的新作品。
  • 标签管理：为番号添加自定义标签（如"经典"、"高分"等），方便后续筛选和整理。
  • 快捷操作：在 JavDB 页面上，鼠标悬停在番号上会显示快速标记按钮，无需打开详情页。
  • 批量处理：在番号库中按住 Ctrl 或 Shift 可多选记录，进行批量标记、删除等操作。

数据管理

强大的本地数据库，让你轻松管理所有观影记录。

📚 番号库

  • 搜索与筛选：支持番号、标题、演员名搜索，可按标签、状态、日期筛选。支持模糊搜索和正则表达式。
  • 批量操作：使用 Ctrl/Shift 多选，批量标记状态、添加标签、删除记录。最多可同时选择 100 条记录。
  • 排序与分组：按添加时间、更新时间、番号、评分排序；支持按状态、标签、演员分组查看。
  • 详情编辑：点击记录可查看详情，编辑标签、备注、评分等信息。支持添加个人笔记和观看日期。
  • 快速标记：使用快捷键 Ctrl+Shift+M 标记"已观看"，Ctrl+Shift+W 标记"想看"。
  • 状态管理：支持"已观看"、"想看"、"想再看"、"已阅"、"跳过"等多种状态，可自定义状态颜色。
  • 评分系统：为每个番号打分（1-5星），方便后续筛选高分作品。
  • 封面预览：鼠标悬停在封面上可查看大图，支持左右切换多张预览图。
  • 磁链管理：查看和管理每个番号的磁力链接，支持一键复制和推送到 115 网盘。
  • 数据导出：可将筛选后的记录导出为 CSV 或 JSON 格式，方便在 Excel 中分析。

💡 番号库使用技巧

  • 智能搜索：输入演员名可搜索该演员的所有作品，输入标签名可搜索带该标签的所有番号。
  • 快速筛选：点击标签或演员名可快速筛选相关记录，再次点击取消筛选。
  • 收藏夹功能：为重要的番号添加"收藏"标签，创建个人精选集。
  • 观看历史：系统自动记录每次标记的时间，可查看观看历史时间线。
  • 重复检测：添加番号时自动检测是否已存在，避免重复收藏。

👤 演员库

  • 收藏同步：自动同步 JavDB 收藏的演员到本地，支持手动刷新。可设置自动同步间隔（每天/每周）。
  • 别名管理：为演员添加自定义别名，方便搜索和识别。支持多个别名，用逗号分隔。
  • 分类整理：创建自定义分类（如"最爱"、"新人"、"退役"等），拖拽排序。支持多级分类。
  • 卡片视图：支持大、中、小三种卡片尺寸，分页浏览。可自定义每页显示数量（12/24/48/96）。
  • 快速跳转：点击演员卡片可直接跳转到 JavDB 演员页面，查看最新作品和详细信息。
  • 演员统计：显示每个演员的作品数量、收藏时间、最新作品发布日期等信息。
  • 新作订阅：订阅演员后，系统会自动追踪并通知新作品发布。
  • 批量管理：支持批量添加标签、移动分类、删除演员等操作。
  • 搜索功能：支持按演员名、别名、分类搜索，支持模糊匹配。
  • 排序选项：按收藏时间、作品数量、最新作品日期、演员名排序。

💡 演员库使用技巧

  • 快速收藏：在 JavDB 演员页面点击扩展添加的"收藏"按钮，即可快速添加到演员库。
  • 分类管理：为不同类型的演员创建分类，如"欧美"、"日本"、"新人"等，方便管理。
  • 别名设置：为演员添加中文名、英文名等别名，搜索时更容易找到。
  • 新作提醒：订阅的演员有新作品时，会在"新作"标签页显示通知。
  • 作品统计：查看每个演员的作品数量和观看进度，了解收藏情况。

🆕 新作发现

  • 自动订阅：订阅关注的演员，自动发现并展示他们的新作品。支持设置检查频率（每天/每周）。
  • 状态联动：新作状态与番号库同步，已标记的作品自动过滤，避免重复提醒。
  • 过滤选项：按演员、发布日期、状态筛选新作。支持只显示未标记的新作。
  • 清理已读：一键清理已标记或已过期的新作通知，保持列表整洁。
  • 发布提醒：新作品发布时显示桌面通知（需授权），第一时间了解动态。
  • 预告片预览：部分新作支持预告片预览，提前了解作品内容。
  • 批量标记：支持批量标记新作为"想看"或"已阅"，快速处理大量通知。
  • 排序方式：按发布日期、添加时间、演员名排序，灵活查看新作。
  • 卡片视图：支持列表和卡片两种显示模式，适应不同浏览习惯。
  • 自动清理：可设置自动清理超过指定天数的已读新作，释放存储空间。

💡 新作发现使用技巧

  • 订阅管理：在演员库中勾选"订阅新作"，系统会自动追踪该演员的新作品。
  • 通知设置：在设置中可配置通知方式（桌面通知/仅面板提示），避免打扰。
  • 快速筛选：点击演员名可快速查看该演员的所有新作，方便集中查看。
  • 发布日历：查看未来一周的新作发布计划，提前做好观看安排。
  • 智能推荐：基于观看历史，系统会推荐可能感兴趣的新作品。

💾 数据导入导出

  • 本地备份：导出为 JSON 文件，保存到本地磁盘。支持压缩导出，减小文件体积。
  • 数据还原：从 JSON 文件导入，恢复历史数据。支持增量导入和完全覆盖两种模式。
  • WebDAV 云备份：自动或手动上传到云端，多设备同步。支持加密备份保护隐私。
  • 选择性备份：可选择备份内容（设置、记录、演员、日志等），灵活控制备份范围。
  • 自动备份：可设置定时自动备份（每天/每周/每月），无需手动操作。
  • 版本管理：保留多个备份版本，可恢复到任意历史版本。
  • 备份验证：导出后自动验证数据完整性，确保备份可用。
  • 导入预览：导入前可预览数据内容和统计信息，避免误操作。
  • 冲突处理：导入时智能检测数据冲突，可选择保留本地或云端数据。
  • 格式转换：支持导出为 CSV、Excel 格式，方便在其他软件中分析。

💡 数据备份最佳实践

  • 定期备份：建议每周至少备份一次，重要数据建议每天备份。
  • 多地备份：同时使用本地备份和 WebDAV 云备份，双重保障数据安全。
  • 验证备份：定期测试备份恢复功能，确保备份文件可用。
  • 版本保留：保留最近 3-5 个备份版本，以防最新备份损坏。
  • 迁移设备：更换设备时，先导出完整备份，在新设备上导入即可。

账户信息

管理你的 JavDB 账户信息，用于数据同步和功能增强。

  • 账户登录：显示并管理你的 JavDB 账户信息（邮箱/用户名/用户类型/会员等级等）。
  • 安全存储：账户凭据仅在本地存储，使用加密技术保护，仅用于与 JavDB 的数据同步。
  • 登录状态：实时显示登录状态和会话有效期，过期时会提示重新登录。
  • 隐私保护：所有数据仅在本地处理，不会上传到第三方服务器，完全保护你的隐私。
  • 多账户支持：支持切换不同的 JavDB 账户，方便管理多个账号的数据。
  • 自动登录：可选择记住登录状态，下次打开自动登录，无需重复输入。
  • 会员特权：显示会员等级和特权信息，充分利用会员功能。
  • 登录历史：查看最近的登录记录和设备信息，及时发现异常登录。

💡 账户安全提示

  • 定期更新：建议定期更新 JavDB 账户密码，保障账户安全。
  • 安全登录：不要在公共设备上勾选"记住登录"，避免账户泄露。
  • 及时退出：长期不使用时建议退出登录，防止他人访问你的数据。
  • 异常检测：如发现异常登录记录，请立即修改密码并检查账户安全。

数据同步（JavDB ↔ 本地）

与 JavDB 网站双向同步你的观看记录和收藏，实现多设备数据一致。

  • 同步全部：一键从 JavDB 获取"已观看 + 想看"的所有记录到本地。首次使用建议执行完整同步。
  • 分类同步：可单独拉取"想看"或"已看"的记录，按需更新。支持增量同步，只拉取新增和更新的记录。
  • 收藏演员：支持同步收藏演员列表，并可单独刷新演员信息和头像。
  • 进度显示：显示"获取列表/拉取详情"两阶段进度，实时显示同步速度和剩余时间，支持随时取消。
  • 增量同步：智能识别新增和更新的记录，避免重复拉取，大幅提升同步速度。
  • 冲突处理：本地与云端数据冲突时，可选择保留哪一方的数据（本地优先/云端优先/手动选择）。
  • 自动同步：可设置定时自动同步（每天/每周），保持数据最新。
  • 同步日志：记录每次同步的详细信息，方便排查问题和查看历史。
  • 断点续传：同步中断后可从断点继续，无需重新开始。
  • 批量上传：支持将本地标记的记录批量上传到 JavDB，实现双向同步。

💡 数据同步最佳实践

  • 首次同步：首次使用建议选择"同步全部"，完整导入 JavDB 的历史数据。
  • 定期同步：建议每周执行一次增量同步，保持本地数据与 JavDB 一致。
  • 网络环境：大量数据同步时建议在 WiFi 环境下进行，避免消耗移动流量。
  • 冲突解决：遇到数据冲突时，建议选择"云端优先"，以 JavDB 数据为准。
  • 同步验证：同步完成后检查统计数据，确认记录数量正确。

WebDAV 备份与恢复

通过 WebDAV 协议实现多设备间的数据同步和备份，支持主流云存储服务。

  • 自动备份：可设置自动备份间隔（如每天、每周、每月），无需手动操作。支持设置备份时间点。
  • 手动备份：随时手动上传当前数据到云端，确保数据安全。支持备份前预览数据大小。
  • 服务兼容：兼容 Nextcloud、Seafile、TeraCloud、Yandex Disk、坚果云等主流 WebDAV 服务。
  • 恢复模式：提供三种恢复模式：
    • 快速替换 - 直接用云端数据覆盖本地，适合数据丢失后的快速恢复
    • 智能合并 - 合并本地和云端数据，保留最新记录，适合多设备同步
    • 专家模式 - 查看详细差异，手动选择保留哪些数据，适合精细控制
  • 选择性备份：可分别备份"扩展设置"、"观看记录"、"账户信息"、"演员库"、"日志"、"新作"等数据模块。
  • 版本管理：支持查看备份历史，恢复到指定版本。自动保留最近 10 个备份版本。
  • 加密选项：可选择加密备份数据（AES-256），保护隐私。需设置加密密码。
  • 压缩备份：自动压缩备份文件，节省云存储空间，压缩率可达 70%。
  • 连接测试：提供连接测试功能，验证 WebDAV 服务器配置是否正确。
  • 同步冲突：智能检测多设备同步冲突，提供冲突解决方案。
  • 带宽控制：可限制上传/下载速度，避免占用过多网络带宽。
  • 断点续传：大文件备份支持断点续传，网络中断后可继续上传。

💡 WebDAV 配置指南

  • Nextcloud：服务器地址格式：https://your-domain.com/remote.php/dav/files/username/
  • 坚果云：需在网页端生成应用密码，不能使用登录密码。地址：https://dav.jianguoyun.com/dav/
  • Seafile：需启用 WebDAV 扩展，地址格式：https://your-domain.com/seafdav/
  • 自建服务：确保服务器支持 HTTPS，避免数据传输被窃听。
  • 路径设置：建议在云盘中创建专门的文件夹存放备份，如 /JavHelper/

💡 WebDAV 使用技巧

  • 首次配置：配置完成后先执行"连接测试"，确认配置正确再进行备份。
  • 加密备份：如果使用公共云服务，建议启用加密备份保护隐私。
  • 定期备份：建议设置每天自动备份，确保数据不丢失。
  • 多设备同步：在多个设备上配置相同的 WebDAV，实现数据自动同步。
  • 版本回退：如果误操作导致数据错误，可从备份历史中恢复到之前的版本。

115 网盘集成

与 115 网盘深度集成，实现一键离线下载和智能管理。

  • 一键推送：在 JavDB 详情页点击按钮，直接将磁力链接推送至 115 离线下载。支持批量推送多个磁链。
  • 验证码处理：自动处理 115 的验证流程，失败可重试。支持图片验证码和滑块验证码。
  • 状态联动：推送成功后可自动标记为"已观看"或其他状态，无需手动操作。
  • 配额显示：启用 115-V2 后，在侧边栏实时显示网盘配额（总量/已用/剩余/使用率）。
  • 任务管理：查看推送历史和下载任务状态（下载中/已完成/失败）。支持重新推送失败的任务。
  • 批量推送：支持批量选择多个磁力链接，一次性推送。可设置推送间隔避免频繁请求。
  • 智能选择：自动识别最优质的磁链（根据文件大小、做种数等），优先推送高质量资源。
  • 目录管理：可设置默认保存目录，自动分类存储不同类型的文件。
  • 下载通知：下载完成后显示桌面通知，及时了解下载进度。
  • 重复检测：推送前自动检测是否已存在相同文件，避免重复下载。
  • 速度限制：可设置推送频率限制，避免触发 115 的反爬机制。

💡 115 网盘配置指南

  • Cookie 获取：在 115 网站登录后，按 F12 打开开发者工具，在 Network 标签中找到请求头的 Cookie 值。
  • V2 版本：115-V2 提供更多功能（配额显示、任务管理等），建议优先使用。

隐私保护

保护你的隐私，避免敏感内容被他人看到，提供多层次的隐私保护方案。

  • 锁屏模式：一键模糊所有敏感区域（封面、标题、演员名、标签等），快速隐藏内容。
  • 自动隐藏：切换标签页后自动启用隐私模式，回到页面时恢复。可设置延迟时间（立即/5秒/10秒）。
  • 模糊强度：可调整模糊程度（轻度/中度/重度/完全隐藏），灵活控制隐私级别。
  • 选择性模糊：选择要模糊的内容类型（封面/标题/演员/标签/评分/备注），精细化控制。
  • 快捷键切换：设置快捷键快速开启/关闭隐私模式，默认 Ctrl+Shift+L。
  • 密码保护：可设置密码，解锁后才能查看内容。支持指纹/面容识别（浏览器支持时）。
  • 无痕模式：启用后不记录浏览历史、搜索记录等敏感信息。
  • 自动清理：关闭浏览器时自动清理缓存、Cookie、历史记录等数据。
  • 伪装模式：将界面伪装成其他类型的网站（如新闻、学习网站），避免被发现。
  • 紧急退出：设置紧急退出快捷键（如 Esc 三连击），快速关闭所有相关标签页。

💡 隐私保护最佳实践

  • 多层防护：同时启用锁屏模式、自动隐藏、密码保护，提供多层次保护。
  • 快捷操作：记住隐私模式快捷键，遇到突发情况可快速隐藏内容。
  • 自动化设置：启用自动隐藏功能，离开页面时自动保护隐私。
  • 定期清理：定期清理浏览历史和缓存数据，避免留下痕迹。
  • 公共场所：在公共场所使用时，建议启用重度模糊或伪装模式。

  • 批量修改：批量修改记录的字段值，如批量更新标签、状态、评分等。
  • SQL 查询：执行自定义 SQL 查询，灵活查询和分析数据（需谨慎使用）。
  • 脚本执行：执行自定义 JavaScript 脚本，实现高级自动化操作。
  • API 测试：测试各个 API 接口的连通性和响应时间，排查网络问题。

💡 高级工具使用提示

  • 备份优先：使用重置功能前，务必先备份当前数据和设置。

常见问题

遇到问题？这里有一些常见问题的解决方案和使用建议。

❓ 数据相关

  • 数据丢失怎么办？可通过 WebDAV 恢复云端备份，或导入之前导出的本地 JSON 备份文件。建议定期备份数据。
  • 如何迁移到新设备？在旧设备导出数据或上传到 WebDAV，在新设备导入或下载即可。也可以配置相同的 WebDAV 实现自动同步。
  • 数据占用空间太大？定期清理无用数据、日志和缓存，或使用"高级工具"中的数据库优化功能。可选择性删除不需要的记录。
  • 数据同步冲突？选择"智能合并"模式可自动解决大部分冲突，复杂情况使用"专家模式"手动处理。
  • 导入数据失败？检查 JSON 文件格式是否正确，确认文件未损坏。可尝试使用"数据校验"功能修复。

🔄 同步相关

  • 同步失败怎么办？检查网络连接和 JavDB 登录状态，查看"日志"获取详细错误信息。可能是网络不稳定或 JavDB 服务器繁忙。
  • WebDAV 连接不上？使用"网络测试"功能检查连接，确认服务器地址、用户名、密码正确。检查是否需要使用应用密码（如坚果云）。
  • 同步速度慢？调整并发数和批次大小，或选择网络状况较好的时间同步。可启用增量同步减少数据传输量。
  • 同步后数据不一致？执行"完整同步"重新拉取所有数据，或使用"数据校验"功能检查数据完整性。
  • WebDAV 备份失败？检查云盘存储空间是否充足，确认 WebDAV 服务正常运行。可尝试减小备份文件大小。

⚙️ 功能相关

  • 115 推送失败？检查 115 账号登录状态和 Cookie 是否过期，确认验证码处理正确。必要时重新获取 Cookie 并更新配置。
  • AI 功能不可用？检查 AI 设置中的 API Key 是否正确，模型是否可用。确认 API 配额未用尽，网络可访问 API 服务器。
  • 标记按钮不显示？刷新页面，或检查"功能增强"设置是否启用。清除浏览器缓存后重试。
  • 搜索引擎跳转失败？检查搜索引擎 URL 配置是否正确，确认使用了 {{ID}} 占位符。测试目标网站是否可访问。
  • Emby 集成不工作？确认 Emby 服务器地址和 API Key 配置正确，检查网络连通性。确保 Emby 版本兼容。
  • 隐私模式无效？检查快捷键是否冲突，确认隐私模式已启用。尝试重新加载页面。

� 性能优化

  • 扩展运行卡顿？关闭不需要的功能，减少同时处理的数据量。清理缓存和日志，执行数据库优化。
  • 大量记录时建议：定期清理无用数据，合理设置并行与阈值参数。使用分页和懒加载减少一次性加载的数据量。
  • 浏览器占用内存高？定期重启浏览器，清理扩展缓存。关闭不使用的标签页和其他扩展。
  • 页面加载慢？启用图片懒加载，减少同时加载的图片数量。关闭不必要的页面增强功能。
  • 数据库查询慢？执行数据库优化，重建索引。减少复杂的筛选和排序操作。

🔧 其他问题

  • 扩展更新后出现问题？清除浏览器缓存，重新加载扩展。检查是否需要数据迁移。
  • 快捷键冲突？在设置中自定义快捷键，避免与其他软件或扩展冲突。
  • 界面显示异常？尝试切换浏览器缩放比例到 100%，清除缓存后刷新页面。
  • 无法登录 JavDB？检查账号密码是否正确，确认 JavDB 网站可正常访问。可能需要更新 Cookie。
  • 功能找不到？使用控制面板的搜索功能，或查看本帮助文档的对应章节。
