快速开始

Jav 助手会在 JavDB 页面上提供状态标记、列表增强、磁链辅助、本地数据管理与同步工具。首次使用建议先完成同步和基础设置。

  • 打开入口：点击扩展图标可打开弹窗；点击“高级设置 & 数据管理”可进入完整控制面板。
  • 首次同步：建议先到完整控制面板的“数据同步”页执行一次同步，把 JavDB 的观看/想看/演员等数据拉到本地。
  • 浏览时自动增强：访问 JavDB 页面后，扩展会按你的设置自动添加状态按钮、列表增强、封面预览、磁链辅助等功能。
  • 集中管理：可在完整控制面板的“首页”“番号库”“演员库”“新作品”“清单”“观影报告”“日志”等页统一管理数据。
  • 按需配置：如果你需要 WebDAV 备份、115 网盘、Emby / Jellyfin 增强、隐私保护或 AI 分析，建议在设置页里完成对应配置。

💡 推荐上手顺序

  • 第 1 步：在“数据同步”页执行“同步全部”。
  • 第 2 步：在“显示过滤”和“功能增强”页打开你常用的浏览增强项。
  • 第 3 步：如果需要长期保存数据，可在“WebDAV同步”页开启自动备份。
  • 第 4 步：如果你使用 115 网盘或 Emby / Jellyfin，可继续配置对应增强功能。
  • 第 5 步：在“隐私保护”页设置密码、自动锁定和恢复方式。

数据管理

完整控制面板提供本地数据库、筛选器、批量工具和多个专用标签页，适合长期整理 JavDB 记录。

📚 番号库

  • 这个页面能做什么：管理本地影片记录，查看概览卡片、筛选记录、编辑标签/备注/评分、做批量处理。
  • 概览卡片：可快速查看总记录、已观看、已浏览、我想看、本周新增、本月新增，并点击卡片直接筛选。
  • 搜索与筛选：支持按番号、标题、标签、清单名称搜索；支持状态筛选、排序、标签筛选和清单筛选。
  • 高级搜索语法：搜索框支持 tag: / tags: / #标签，也支持 list: / listid: 语法。
  • 视图模式：支持列表视图 / 卡片视图切换，并可显示或隐藏封面。
  • 收藏与编辑：支持“我的收藏”、星级评分、备注、标签和清单归类。
  • 批量操作：支持批量改状态、批量加标签、批量修改清单、批量刷新、批量删除。
  • 导出数据：可从番号库直接导出本地记录。

💡 常用操作

  • 按标签筛选：在搜索框输入 tag:中出、tags:素人,人妻 或 #人妻。
  • 按清单筛选：输入 list:精选 或 listid:xxxxxx，会和清单筛选联动。
  • 只看某个状态：点击概览卡片或使用上方状态筛选器。
  • 批量整理：先多选记录，再使用批量操作菜单统一修改状态、标签或清单。

🎭 演员库

  • 这个页面能做什么：管理演员资料、收藏状态、黑名单、订阅状态和分类。
  • 概览卡片：显示全部演员、女演员、男演员、有码、无码、黑名单、本周新增等统计。
  • 搜索与筛选：可按姓名/别名、性别、分类、状态、排序方式筛选。
  • 批量工具：支持批量刷新元数据、批量拉黑/取消拉黑、批量订阅/取消订阅、批量删除。
  • 视图模式：支持列表和卡片模式切换。
  • 页面联动：可配合“演员页增强”功能，在站点页面做时间分段、新作品扫描和快速操作。

🆕 新作品

  • 这个页面能做什么：集中查看订阅演员的新作品，并按未读、今日发现等维度快速整理。
  • 概览卡片：支持按订阅演员总数、活跃订阅、全部新作品、未读作品、今日发现进行快速筛选。
  • 订阅追踪：基于已订阅演员追踪新作品，并在面板中集中查看。
  • 未读管理：支持未读筛选、已读状态切换和列表整理。
  • 视图切换：支持列表 / 卡片视图切换，便于快速浏览。

🗂️ 清单

  • 这个页面能做什么：管理本地清单，并把清单作为番号库的辅助分类工具。
  • 本地清单：可新建本地清单、刷新清单列表、搜索清单。
  • 跳转联动：点击清单可跳转到番号库，并自动带上对应的清单筛选条件。
  • 与番号库联动：番号库支持按清单筛选，也支持批量把记录加入或移出清单。

🏠 首页

  • 这个页面能做什么：查看总览、趋势和标签排行，快速判断当前数据规模与变化。
  • 概览与趋势：首页提供总记录、演员、新作品等概览信息和趋势图表。
  • 时间范围：支持按预设时间范围或自定义日期区间查看趋势。
  • 标签排行：支持查看热门标签并翻页浏览。

数据同步（JavDB ↔ 本地）

“数据同步”页用于把 JavDB 的状态、演员与相关数据导入本地数据库，便于后续在完整控制面板中做筛选、统计和备份。

  • 这个页面能做什么：把 JavDB 的历史状态同步到本地，并补齐控制面板各页的数据来源。
  • 同步类型：支持同步全部、想看、已观看、演员、演员性别等数据。
  • 两段进度：同步过程会显示“获取列表页面”和“获取视频详情”两段进度。
  • 支持取消：同步过程中可以随时取消。
  • 同步日志：失败原因和进度信息会写入“日志”页，便于排查。
  • 结果联动：同步结果会进入番号库、演员库、首页和观影报告统计。

💡 常用操作

  • 第一次使用：建议先执行“同步全部”，后续再按需执行专项同步。
  • 演员信息缺失：如果演员分类或性别数据不完整，可单独执行演员同步或演员性别同步。
  • 同步后没看到结果：优先去“日志”页看失败信息，再检查 JavDB 登录状态和网络情况。

WebDAV 备份与恢复

“WebDAV同步”页用于将本地数据备份到兼容的 WebDAV 存储，并提供恢复、回滚、下载备份等工具。

  • 这个页面能做什么：管理 WebDAV 配置、自动备份、手动备份、恢复与诊断。
  • 多配置管理：支持添加多个 WebDAV 配置，并测试或诊断连接。
  • 自动上传：支持启用自动上传、自定义同步间隔、备份保留数量和预警阈值。
  • 手动备份与恢复：支持手动创建备份、查看备份文件、下载最新备份、从云端恢复。
  • 恢复前预览：恢复流程要求先分析并预览备份内容，再执行恢复。
  • 恢复前保护：恢复时会先创建恢复前备份，便于出现问题时回滚。
  • 本地与云端配合：建议把 WebDAV 备份和本地导出一起使用，降低单点损坏风险。

💡 常用操作

  • 首次配置：先添加配置，再点击“测试连接”或“诊断连接”。
  • 开始自动备份：启用 WebDAV，再打开“自动上传”，设置同步间隔和保留数量。
  • 恢复前检查：恢复前认真看预览内容；如果不确定，先下载最新备份留档。

115 网盘集成

115 相关功能分为“115网盘”设置页和“115网盘下载任务”页两部分，可用于添加云下载、设置下载目录、查看任务和导出日志。

  • 这个页面能做什么：配置 115-V2、添加任务、测试搜索、查看任务状态和日志。
  • 115-V2 配置：支持启用 115-V2、填写 Cookie、设置根目录 ID 和下载目录 ID。
  • 添加云下载任务：可在设置页直接添加 URL / 磁链任务，也可在“115网盘下载任务”页手动输入链接。
  • 任务页：提供任务列表、搜索、刷新、清空和分页浏览。
  • 下载设置：支持设置验证次数、最大失败数、自动通知等参数。
  • 测试与日志：支持测试搜索、查看 115 专用日志、刷新 / 清空 / 导出日志。
  • 与番号库联动：在番号库或影片页增强功能中，可结合磁链管理使用 115 推送。

💡 页面区别

  • 115网盘设置页：用于配置 Cookie、目录、验证参数和测试功能。
  • 115网盘下载任务页：用于查看当前任务、搜索任务、刷新任务和清空任务列表。

Emby 增强

“Emby增强设置”页可让 Emby / Jellyfin 页面自动识别番号，并提供跳转到 JavDB 的快捷入口。

  • 这个页面能做什么：配置要增强的网址、点击跳转行为和右侧快捷按钮。
  • 网址匹配：支持自定义网址模式，使用通配符匹配要启用增强的页面。
  • 链接行为：点击识别出的番号后，可跳转到 JavDB 搜索页或直接跳转详情页。
  • 快捷按钮：支持显示“搜番号”“搜演员”悬浮按钮。
  • 适用场景：适用于 Emby、Jellyfin 等能显示文本番号的媒体库页面。

显示与内容过滤

“显示过滤”页和弹窗控制项会直接作用在 JavDB 页面与扩展界面中，帮助你隐藏不想看的内容或快速调整浏览方式。

  • 这个页面能做什么：控制隐藏规则、演员过滤和弹窗中的显示参数。
  • 影片过滤：支持隐藏已观看、已浏览、VR、我想看的影片。
  • 演员过滤：支持隐藏包含黑名单演员、未收藏演员、无法识别演员的作品，并可将“订阅”视为“收藏”。
  • 弹窗控制：弹窗中支持调整详情页预览视频音量、列表列数和容器宽度。
  • 快速锁定：如果已启用隐私保护，可在弹窗中快速执行锁定相关操作。

搜索引擎跳转

“搜索引擎设置”页用于配置点击番号后跳转的目标站点，可自定义多个搜索模板。

  • 这个页面能做什么：管理外部搜索站点列表，决定番号点击后的跳转目标。
  • 自定义引擎：支持添加、编辑、删除搜索引擎。
  • URL 模板：使用 {{ID}} 作为番号占位符。
  • 图标配置：可为每个搜索引擎配置图标和图标地址。
  • 页面联动：结合列表页和影片页的增强入口，可快速跳转外部站点搜索资源或信息。

功能增强

“功能增强设置”页包含大量分站点、分页面的开关，用于控制列表页、影片页、演员页和其他增强项。

  • 这个页面能做什么：选择你希望扩展在 JavDB 页面自动执行的增强功能。
  • 列表页增强：可启用列表状态显示、容器宽度控制、点击增强、滚动翻页等功能。
  • 影片页增强：可启用磁链搜索、评论区突破、FC2 相关突破、演员快速操作、收藏评分等功能。
  • 演员页增强：支持演员页增强、自动应用过滤器、默认过滤条件、时间分段、新作品扫描等。
  • 其他增强：包括密码辅助、锚点优化、内容辅助等页面增强项（以设置页实际开关为准）。
  • 编排可视化：可查看当前页面启用了哪些功能编排，方便排查冲突。

隐私保护

“隐私保护”页用于在本地对扩展界面和页面内容进行模糊、锁定与密码保护。

  • 这个页面能做什么：管理模糊规则、密码、自动锁定和恢复方式。
  • 模糊模式：支持开启内容模糊，隐藏敏感内容。
  • 锁定与密码：支持设置密码、更改密码、取消密码，并配置无操作超时自动锁定。
  • 自动锁定：支持离开标签页自动锁定、关闭扩展自动锁定。
  • 密码恢复：支持设置安全问题和生成备份恢复码。
  • 兜底恢复：如果没有恢复方式且忘记密码，系统会提示只能重置数据，因此建议至少保留一种恢复方式。

💡 常用操作

  • 启用密码保护：先设置密码，再打开需要的自动锁定选项。
  • 避免忘记密码：至少配置“安全问题”或“备份恢复码”中的一种。
  • 临时离开屏幕：可用快捷锁定方式快速进入保护状态。

快捷键

扩展内置一组页面快捷键，适合在 JavDB 站点中快速导航、复制信息和切换过滤。

  • 导航：Alt + H 返回首页，Alt + S 打开搜索页，/ 聚焦搜索框。
  • 滚动：Home 回到顶部，End 跳到底部。
  • 复制：Ctrl + Shift + V 复制番号，Ctrl + Shift + T 复制标题，Ctrl + Shift + U 复制页面链接。
  • 过滤：Alt + V 切换已看过滤，Alt + R 切换 VR 过滤，Alt + Shift + R 重置过滤器。
  • 帮助：? 显示 / 隐藏快捷键帮助面板。
  • 隐私相关：隐私模块还支持页面上的锁定快捷操作，若无效请检查是否启用了隐私保护。

AI 功能

“AI 功能”页主要用于对话测试、模型连接验证，以及为“观影报告”提供 AI 增强分析。

  • 这个页面能做什么：配置 AI 服务、测试连接、调节模型参数，并把 AI 用于报告分析。
  • 兼容接口：支持 OpenAI 兼容接口或 One API 类兼容服务。
  • 基础配置：可配置 API 地址、API Key、模型、温度、最大 Token、超时时间和系统提示词。
  • 连接测试：设置页内可直接测试连接和发送测试消息。
  • 报告增强：在“观影报告”页中，如启用 AI，可为月报 / 区间报告生成更丰富的文本分析。
  • 按需启用：未配置 AI 时，“观影报告”仍可生成基础版 HTML 报告。

统计与概览

“首页”和“观影报告”页提供本地数据的统计可视化，适合做月度回顾和长期趋势分析。

  • 首页趋势：支持查看记录趋势、演员趋势、新作品趋势、状态分布和标签排行。
  • 时间范围：首页支持预设范围和自定义时间区间。
  • 观影报告：支持按月份范围生成报告、预览历史、刷新历史、删除所选历史。
  • 报告导出：当前支持导出 HTML、Markdown、JSON。
  • 自动生成：“观影报告设置”页提供自动生成和相关阈值配置。

💡 常用操作

  • 生成一份报告：到“观影报告”页选择月份范围，点击“生成报告”。
  • 导出当前预览：若未勾选历史记录，可直接导出当前预览的 HTML / Markdown。
  • 导出历史记录：先勾选历史项，再导出 HTML / Markdown / JSON。
  • 看不到内容：优先确认选择区间内是否有足够的数据样本。

日志与诊断

“日志”页和相关诊断工具用于定位同步失败、网络异常、115 下载问题和其他运行问题。

  • 日志页：支持查看扩展持久化日志和控制台即时日志。
  • 过滤检索：支持按关键词、级别、类别、日期范围、是否包含详细数据进行筛选。
  • 日志操作：支持刷新、清空、导出日志，支持调整每页显示数量。
  • 115 专用日志：“115网盘”设置页单独提供 115 日志查看、清空、导出。
  • WebDAV 诊断：“WebDAV同步”页提供连接测试与诊断。
  • 网络测试：“网络测试”页支持手动 URL 延迟测试、批量域名测试、核心域名测试、域名配置和线路管理。

💡 常用操作

  • 同步失败排查：先到“日志”页搜错误关键词，再看对应类别和详细数据。
  • 网络异常排查：到“网络测试”页先做单个 URL 测试，再做批量域名测试。
  • JavDB 线路问题：可在“网络测试”页测试所有线路、从 GitHub 更新线路，或恢复默认线路。

高级工具

“高级工具”页集中放置原始设置、原始日志和全局危险操作，适合排障或迁移时使用。

  • 原始设置：支持查看、编辑和导出完整备份 JSON。
  • 原始日志：支持查看原始日志并写入测试日志。
  • 全局操作：支持清空所有本地记录、清空缓存、清空临时数据、重置所有设置、重新加载扩展。
  • 版本与更新：“版本更新”页提供当前版本、最新版本、发布信息预览与检查更新。
  • 线路与代理：“网络测试”页可管理 GitHub 加速服务、JavDB 线路和批量连通性测试。

常见问题

下面列出使用中最常见的几类问题，以及对应的排查方向。

🔐 同步 / 备份问题

  • 同步失败：先确认 JavDB 登录状态和网络环境，再查看“日志”页中的错误详情。
  • WebDAV 连接失败：优先使用“测试连接”与“诊断连接”，检查地址、账号、密码和路径。
  • 恢复不放心：先做预览；恢复流程会先创建恢复前备份，必要时可回滚。

🎬 页面增强问题

  • 增强按钮没出现：确认对应增强开关已开启，并刷新当前页面。
  • 磁链 / 搜索功能异常：检查“功能增强设置”和“搜索引擎设置”是否已正确配置。
  • Emby 增强无效：确认当前网址命中了你配置的模式。

🧠 AI / 报告问题

  • AI 测试失败：检查 API 地址、密钥、模型和网络是否可用。
  • 报告生成为空：确认所选时间范围内有足够的数据记录。
  • 导出格式：当前支持 HTML / Markdown / JSON，不要按 PDF 使用。

🛡️ 隐私问题

  • 忘记密码：优先使用安全问题或备份恢复码；若都未配置，可能只能重置数据。
  • 自动锁定不生效：检查无操作超时、离开标签页锁定、关闭扩展锁定是否开启。

💬 需要帮助？

如果仍然无法判断问题原因，建议按下面顺序排查：

  • 先查看“日志”页里的错误信息
  • 必要时导出日志或备份，保留现场
  • 再结合“WebDAV同步”“115网盘”“AI 功能”“网络测试”等设置页逐项排查
  • 若问题稳定可复现，再携带日志、截图和操作步骤反馈
