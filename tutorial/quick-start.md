# 快速开始

欢迎使用 Jav 助手！本指南将帮助你快速安装和配置扩展。

## 安装扩展

### 方式一：下载预编译版本（推荐）

1. 访问 [GitHub Releases 页面](https://github.com/Adsryen/JavdBviewed/releases)
2. 下载最新版本的 `javdb-extension-v*.zip` 文件
3. 解压到本地文件夹（建议保存在固定位置，避免误删）
4. 打开浏览器扩展管理页面：
   - Chrome: 地址栏输入 `chrome://extensions/`
   - Edge: 地址栏输入 `edge://extensions/`
5. 开启右上角的"开发者模式"开关
6. 点击"加载已解压的扩展程序"按钮
7. 选择刚才解压的文件夹

### 方式二：从源码构建

如果你想使用最新的开发版本或进行二次开发：

```bash
# 克隆项目
git clone https://github.com/Adsryen/JavdBviewed.git
cd JavdBviewed

# 安装依赖
npm install

# 构建扩展
npm run build

# 构建完成后，在浏览器中加载 dist 文件夹
```

## 首次配置

### 1. 打开设置面板

安装完成后，点击浏览器工具栏中的扩展图标，然后点击"设置"按钮进入设置面板。

### 2. 基本设置

在"基本设置"选项卡中，你可以配置：

- **自动标记已浏览**: 访问视频详情页时自动标记为"已浏览"（默认开启）
- **显示标记按钮**: 在列表页和详情页显示快捷标记按钮（默认开启）
- **Favicon 状态指示**: 根据视频状态改变浏览器标签图标（默认开启）

### 3. 开始使用

配置完成后，访问 JavDB 网站，你会看到：

- 列表页的视频卡片上显示标记按钮
- 已浏览过的视频会有灰色标记
- 详情页顶部显示快速操作按钮

## 基本操作

### 标记视频状态

有三种视频状态可以标记：

1. **已观看** (绿色勾选) - 表示你已经看过这个视频
2. **想看** (黄色星标) - 表示你对这个视频感兴趣，但还没看
3. **已浏览** (灰色标记) - 自动标记，表示你访问过详情页

#### 在详情页标记

1. 打开任意视频详情页
2. 页面顶部会显示快捷操作按钮
3. 点击"已观看"或"想看"按钮即可标记

#### 在列表页标记

1. 浏览视频列表
2. 鼠标悬停在视频卡片上
3. 点击出现的标记按钮

#### 使用快捷键

- 按 `V` 键：标记为已观看
- 按 `W` 键：标记为想看

## 查看数据

### 打开设置面板

点击扩展图标，然后点击"设置"按钮，你可以看到：

- **首页**: 数据统计和可视化图表
- **番号库**: 查看所有已标记的视频
- **演员库**: 管理收藏的演员
- **新作品**: 查看订阅演员的新作品

### 番号库

在番号库中，你可以：

- 查看所有已标记的视频列表
- 按状态筛选（已观看/想看/已浏览）
- 搜索特定番号
- 批量操作（删除、修改状态）

## 常见问题

### 扩展安装后没有反应？

1. 确保在扩展管理页面开启了"开发者模式"
2. 检查扩展是否已启用
3. 刷新 JavDB 页面

### 标记按钮没有显示？

1. 检查设置中是否启用了"显示标记按钮"
2. 确保你访问的是 JavDB 网站
3. 尝试刷新页面

### 数据会丢失吗？

扩展的数据存储在浏览器本地，不会自动丢失。但建议：

1. 定期导出数据备份
2. 配置 WebDAV 云端同步
3. 不要随意清除浏览器数据
