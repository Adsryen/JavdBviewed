## 💡 使用技巧

- 💾 **定期备份**: 在"数据管理"中导出数据，避免数据丢失
- ☁️ **启用同步**: 配置 WebDAV 实现多设备数据同步
- 🎯 **智能过滤**: 使用演员黑名单和内容过滤提升浏览体验
- ⚡ **快捷键**: 熟悉键盘快捷键提高操作效率
- 🔒 **隐私保护**: 启用截图模糊模式保护个人隐私
