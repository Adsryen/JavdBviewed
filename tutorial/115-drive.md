# 115网盘集成

115网盘集成功能让你可以直接从视频详情页推送磁力链接到115网盘，实现一键下载。

## 功能概述

- 一键推送磁力链接到115网盘
- 自动处理验证码
- 推送成功后自动标记为已观看
- 批量推送支持
- 任务管理和状态监控

## 前置要求

### 1. 115网盘账号

- 需要有115网盘账号（免费或付费均可）
- 建议使用付费账号，下载速度更快

### 2. 登录115网盘

在使用推送功能前，必须先登录115网盘：

1. 在浏览器中打开 https://115.com
2. 登录你的115账号
3. 保持登录状态

⚠️ **重要**: 扩展需要使用你的115登录状态，请确保已登录

## 启用功能

### 1. 打开设置

1. 点击扩展图标
2. 点击"设置"按钮
3. 进入"115网盘"选项卡

### 2. 启用115功能

1. 开启"启用115网盘功能"开关
2. 功能启用后，详情页会显示推送按钮

### 3. 配置下载目录

设置磁力链接的下载目录：

1. 在"下载目录"输入框中输入路径
2. 格式：`/视频/JAV/` （以 `/` 开头和结尾）
3. 如果目录不存在，会自动创建

**推荐目录结构**：
```
/视频/
  └─ JAV/
      ├─ 有码/
      ├─ 无码/
      └─ 欧美/
```

## 使用推送功能

### 单个推送

#### 方式一：详情页推送按钮

1. 打开视频详情页
2. 找到磁力链接区域
3. 点击磁力链接旁的"推送115"按钮
4. 等待推送完成

#### 方式二：右键菜单

1. 右键点击磁力链接
2. 选择"推送到115网盘"
3. 等待推送完成

### 批量推送

在视频详情页推送多个磁力链接：

1. 选择要推送的磁力链接
2. 点击"批量推送"按钮
3. 扩展会依次推送所有链接
4. 显示推送进度

### 推送流程

1. **提交任务**: 点击推送按钮
2. **验证登录**: 检查115登录状态
3. **处理验证码**: 如果需要，自动处理验证码
4. **推送磁力**: 将磁力链接添加到115
5. **完成标记**: 推送成功后自动标记视频

## 验证码处理

### 自动验证

扩展会自动处理115的验证码：

1. 检测到验证码时自动弹出
2. 完成验证后自动继续推送
3. 无需手动干预

### 手动验证

如果自动验证失败：

1. 会弹出验证窗口
2. 手动完成验证
3. 点击"继续"按钮
4. 推送会自动继续

### 验证失败

如果验证多次失败：

1. 检查网络连接
2. 刷新115网盘页面
3. 重新登录115账号
4. 重试推送

## 自动标记

### 启用自动标记

推送成功后自动标记视频为"已观看"：

1. 在"115网盘"设置中
2. 开启"推送成功后自动标记为已观看"
3. 推送成功后会自动标记

### 标记规则

- 只有推送成功才会标记
- 如果视频已经是"想看"状态，会改为"已观看"
- 如果推送失败，不会标记

### 关闭自动标记

如果不想自动标记：

1. 关闭"推送成功后自动标记为已观看"开关
2. 推送成功后不会改变视频状态

## 任务管理

### 查看任务列表

1. 打开设置面板
2. 进入"115任务"选项卡
3. 查看所有推送任务

### 任务状态

- **等待中**: 任务已创建，等待执行
- **推送中**: 正在推送到115
- **验证中**: 正在处理验证码
- **成功**: 推送成功
- **失败**: 推送失败

### 任务操作

#### 重试失败任务

1. 找到失败的任务
2. 点击"重试"按钮
3. 任务会重新执行

#### 取消任务

1. 找到等待中或推送中的任务
2. 点击"取消"按钮
3. 任务会被取消

#### 删除任务

1. 选择要删除的任务
2. 点击"删除"按钮
3. 任务记录会被删除

#### 清空任务

1. 点击"清空已完成"按钮
2. 所有成功的任务会被清除
3. 失败的任务会保留

### 任务统计

在任务列表顶部显示：

- **总任务数**: 所有任务数量
- **成功数**: 推送成功的任务
- **失败数**: 推送失败的任务
- **成功率**: 成功任务的百分比

## 推送通知

### 通知类型

- **推送成功**: 显示绿色通知
- **推送失败**: 显示红色通知
- **需要验证**: 显示黄色通知

### 通知设置

1. 在"115网盘"设置中
2. 可以开启/关闭推送通知
3. 可以设置通知显示时长

## Cookie 管理

### 自动管理

扩展会自动管理115的登录状态：

- 自动获取登录 Cookie
- 自动刷新过期 Cookie
- 保持登录状态

### 手动刷新

如果登录状态失效：

1. 在"115网盘"设置中
2. 点击"刷新登录状态"按钮
3. 重新获取 Cookie

⚠️ **注意**: 目前 refreshtoken 自动刷新机制还在优化中，如果遇到登录失效，请手动重新登录115网盘

### 清除 Cookie

如果需要切换账号：

1. 点击"清除登录信息"按钮
2. 在115网盘重新登录
3. 刷新登录状态

## 高级设置

### 推送超时

设置推送操作的超时时间：

- **30秒**: 快速超时，适合网络好的情况
- **60秒**: 推荐设置
- **120秒**: 网络较慢时使用

### 重试次数

设置推送失败后的重试次数：

- **0次**: 不重试
- **1-3次**: 推荐设置
- **5次**: 网络不稳定时使用

### 并发推送

设置同时推送的任务数：

- **1个**: 串行推送，最稳定
- **2-3个**: 推荐设置，平衡速度和稳定性
- **5个**: 快速推送，可能不稳定

## 故障排查

### 推送失败

#### 检查登录状态

1. 打开 https://115.com
2. 确认已登录
3. 刷新扩展的登录状态

#### 检查网络

1. 确认能正常访问115网盘
2. 检查是否需要代理
3. 尝试在115网盘手动添加任务

#### 检查磁力链接

1. 确认磁力链接格式正确
2. 磁力链接必须以 `magnet:?` 开头
3. 尝试复制链接到115网盘手动添加

### 验证码无法通过

1. 检查网络连接
2. 尝试手动完成验证
3. 刷新页面后重试
4. 清除浏览器缓存

### Cookie 失效

1. 重新登录115网盘
2. 点击"刷新登录状态"
3. 如果还是失效，清除登录信息后重新登录

### 推送成功但未标记

1. 检查是否启用了"自动标记"
2. 查看浏览器控制台是否有错误
3. 手动标记视频状态

## 使用技巧

### 快速推送工作流

1. 在列表页标记"想看"
2. 从"想看"列表打开视频
3. 查看磁力链接质量
4. 选择最佳链接推送
5. 推送成功自动标记为"已观看"

### 批量下载

1. 打开多个视频详情页
2. 在每个页面点击推送
3. 任务会自动排队执行
4. 在任务列表监控进度

### 选择最佳磁力

推送前检查磁力链接质量：

- **文件大小**: 越大通常质量越好
- **发布时间**: 越新越好
- **做种数**: 越多下载越快
- **文件名**: 包含清晰度信息（1080p、4K等）

### 目录管理

建议按分类创建目录：

```
/视频/JAV/
  ├─ 2024/          # 按年份
  ├─ 有码/          # 按类型
  ├─ 无码/
  ├─ 演员名/        # 按演员
  └─ 系列/          # 按系列
```

## 安全提示

### 隐私保护

- 扩展不会上传你的115账号信息
- Cookie 只存储在本地浏览器
- 不会记录你的下载内容

### 账号安全

- 不要在公共电脑上使用
- 定期修改115密码
- 启用115的两步验证

### 合规使用

- 遵守当地法律法规
- 不要下载违法内容
- 尊重版权

## 常见问题

### 为什么推送失败？

最常见的原因：
1. 115账号未登录
2. 磁力链接无效
3. 网络连接问题
4. Cookie 已过期

### 推送成功但115没有任务？

1. 检查115的"传输列表"
2. 可能任务在"已完成"中
3. 刷新115页面

### 可以推送到指定目录吗？

可以，在设置中配置"下载目录"即可。

### 支持其他网盘吗？

目前只支持115网盘，其他网盘功能计划中。
