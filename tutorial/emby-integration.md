# Emby 集成

Emby 集成功能让你可以将 JavDB 与 Emby 媒体服务器连接，实现快速跳转和状态同步。

## 功能概述

- 配置 Emby 服务器连接
- 自动匹配 JavDB 和 Emby 中的视频
- 从 JavDB 快速跳转到 Emby 播放
- 同步观看状态
- Emby 页面增强功能

## 配置 Emby 服务器

### 1. 获取 Emby 信息

首先需要获取 Emby 服务器信息：

#### 服务器地址

- 本地服务器: `http://localhost:8096`
- 远程服务器: `http://your-server-ip:8096`
- HTTPS: `https://your-domain.com`

#### API 密钥

1. 登录 Emby 管理后台
2. 进入"设置" → "高级" → "API 密钥"
3. 点击"新建 API 密钥"
4. 输入应用名称（如：Jav助手）
5. 复制生成的 API 密钥

### 2. 在扩展中配置

1. 打开设置面板
2. 进入"Emby集成"选项卡
3. 填写配置信息：
   - **服务器地址**: Emby 服务器 URL
   - **API 密钥**: 刚才生成的密钥
   - **用户ID**: Emby 用户 ID（可选）

### 3. 测试连接

1. 点击"测试连接"按钮
2. 等待测试结果
3. 显示"连接成功"说明配置正确

## 自动匹配

### 匹配规则

扩展会自动匹配 JavDB 和 Emby 中的视频：

#### 按番号匹配

- 从文件名中提取番号
- 与 JavDB 的番号对比
- 支持多种番号格式

#### 按标题匹配

- 对比视频标题
- 支持模糊匹配
- 处理标题中的特殊字符

### 匹配设置

配置匹配参数：

- **匹配模式**: 严格/宽松
- **番号格式**: 支持的番号格式
- **标题相似度**: 匹配阈值（0-100）

## 快速跳转

### 从 JavDB 跳转到 Emby

在 JavDB 视频详情页：

1. 如果视频在 Emby 中存在
2. 会显示"在 Emby 中播放"按钮
3. 点击按钮跳转到 Emby 播放页面

### 从 Emby 跳转到 JavDB

在 Emby 视频页面：

1. 扩展会识别番号
2. 显示"在 JavDB 中查看"按钮
3. 点击跳转到 JavDB 详情页

## 状态同步

### 观看状态同步

同步观看状态到 Emby：

1. 在设置中开启"状态同步"
2. 在 JavDB 标记为"已观看"
3. 自动同步到 Emby 标记为"已播放"

### 同步方向

选择同步方向：

- **单向同步**: 只从 JavDB 同步到 Emby
- **双向同步**: JavDB 和 Emby 互相同步
- **手动同步**: 手动触发同步

### 同步设置

配置同步参数：

- **自动同步**: 标记后自动同步
- **同步间隔**: 定期同步的时间间隔
- **冲突处理**: 数据冲突时的处理方式

## Emby 页面增强

### 右侧快捷按钮

在 Emby 页面显示快捷操作按钮：

1. 开启"Emby 页面增强"
2. 在 Emby 视频页面右侧显示按钮
3. 包含：
   - 跳转到 JavDB
   - 标记观看状态
   - 添加到收藏

### 视频信息增强

增强 Emby 中的视频信息显示：

1. 显示 JavDB 评分
2. 显示演员信息
3. 显示标签信息
4. 显示相关推荐

### 播放控制

增强的播放控制功能：

- 快捷键控制
- 播放进度同步
- 字幕设置

### 媒体库管理

管理 Emby 媒体库：

- 查看媒体库列表
- 扫描新内容
- 刷新元数据

## 番号识别

### 识别范围

扩展会在 Emby 页面识别番号：

- **视频标题**: 从标题中提取番号
- **文件名**: 从文件名中提取番号
- **评论区**: 识别评论中的番号

⚠️ **待优化**: 符合 Emby 页面的番号识别功能

### 高亮显示

识别到的番号会高亮显示：

1. 开启"番号高亮"功能
2. 番号会以特殊颜色显示
3. 点击番号跳转到 JavDB

### 高亮样式自定义

自定义番号高亮显示样式：

1. 在设置中找到"高亮样式"
2. 选择颜色和样式
3. 实时预览效果

**样式选项**:
- **颜色**: 高亮颜色
- **背景**: 背景颜色
- **下划线**: 是否显示下划线
- **加粗**: 是否加粗显示

### 链接行为设置

设置点击番号后的跳转行为：

- **直接跳转**: 直接打开 JavDB 详情页
- **搜索**: 在 JavDB 搜索该番号
- **弹窗预览**: 在弹窗中预览信息

## 故障排查

### 连接失败

检查：

1. 服务器地址是否正确
2. API 密钥是否有效
3. 网络连接是否正常
4. 防火墙是否阻止

### 匹配不到视频

可能原因：

1. 文件名格式不标准
2. 番号提取失败
3. 匹配规则太严格
4. 调整匹配设置

### 跳转按钮不显示

检查：

1. 是否启用了 Emby 集成
2. 视频是否匹配成功
3. 刷新页面重试
4. 查看控制台错误

### 状态同步失败

检查：

1. 是否启用了状态同步
2. API 权限是否足够
3. 网络连接是否正常
4. 查看同步日志

## 常见问题

### 需要 Emby 付费版吗？

基本功能免费版即可使用，高级功能可能需要 Emby Premiere。

### 支持 Jellyfin 吗？

目前只支持 Emby，Jellyfin 支持计划中。

### 会修改 Emby 数据吗？

只有启用状态同步时才会修改观看状态，其他数据不会修改。

### 可以连接多个 Emby 服务器吗？

目前只支持一个服务器，多服务器支持计划中。
