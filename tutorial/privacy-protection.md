# 隐私保护

隐私保护功能帮助你在截图或分享时保护敏感信息，避免隐私泄露。

## 功能概述

- 一键模糊敏感内容
- 自动检测截图工具
- 可调节模糊强度
- 自定义保护区域
- 快捷键快速切换

## 启用隐私保护

### 1. 打开设置

1. 打开设置面板
2. 进入"隐私保护"选项卡

### 2. 配置保护模式

1. 开启"启用隐私保护"
2. 配置保护级别
3. 设置快捷键

## 截图模糊模式

### 一键模糊

快速模糊所有敏感内容：

1. 按快捷键（默认 `P` 键）
2. 或点击页面上的"隐私模式"按钮
3. 所有敏感内容立即模糊

### 自动模糊

自动检测截图工具并模糊：

1. 在设置中开启"自动检测截图"
2. 当检测到截图工具时自动模糊
3. 截图完成后自动恢复

**支持的截图工具**:
- Windows 截图工具
- Snipping Tool
- ShareX
- 其他常见截图软件

### 快捷键触发

使用快捷键快速切换：

- **默认快捷键**: `P` 键
- **自定义快捷键**: 在设置中修改
- **切换模式**: 再次按快捷键恢复

### 模糊强度调节

调节模糊程度：

1. 在设置中找到"模糊强度"
2. 拖动滑块调整（0-100）
3. 实时预览效果

**推荐值**:
- **低（20-40）**: 轻微模糊，仍可辨认
- **中（40-60）**: 推荐设置，平衡效果
- **高（60-100）**: 完全模糊，无法辨认

## 元素保护

### 封面模糊

模糊视频封面图片：

1. 开启"模糊封面"选项
2. 所有封面图片会被模糊
3. 包括列表页和详情页

### 标题模糊

模糊视频标题文字：

1. 开启"模糊标题"选项
2. 视频标题会被模糊或替换为 `***`
3. 保护标题中的敏感词

### 演员名模糊

模糊演员名称：

1. 开启"模糊演员名"选项
2. 演员名称会被模糊
3. 包括列表页和详情页

### 番号模糊

模糊视频番号：

1. 开启"模糊番号"选项
2. 番号会被模糊或替换
3. 保护番号信息

### 自定义保护区域

添加自定义需要保护的元素：

1. 点击"添加保护区域"
2. 使用选择器工具选择元素
3. 或手动输入 CSS 选择器
4. 保存配置

**示例选择器**:
```css
.video-title
.actor-name
.video-code
```

## 隐私设置

### 默认模糊模式

设置默认是否启用模糊：

1. 开启"默认启用模糊"
2. 每次打开页面自动模糊
3. 适合经常需要截图的场景

### 模糊区域配置

配置需要模糊的页面区域：

- **列表页**: 封面、标题、演员名
- **详情页**: 封面、标题、演员名、番号、简介
- **演员页**: 演员头像、演员名、作品封面
- **设置页**: 统计数据、记录列表

### 快捷键自定义

自定义隐私保护快捷键：

1. 在设置中找到"快捷键设置"
2. 点击"隐私模式快捷键"
3. 按下新的快捷键
4. 保存配置

**推荐快捷键**:
- `P` - Privacy（隐私）
- `H` - Hide（隐藏）
- `Ctrl+Shift+P` - 组合键

### 保护级别

设置不同的保护级别：

#### 低级别
- 只模糊封面
- 适合一般场景

#### 中级别（推荐）
- 模糊封面和标题
- 平衡保护和可用性

#### 高级别
- 模糊所有敏感信息
- 最大程度保护隐私

## 使用场景

### 截图分享

需要截图分享时：

1. 按 `P` 键启用隐私模式
2. 进行截图
3. 再按 `P` 键恢复正常

### 录制视频

录制教程或演示视频时：

1. 开启"默认启用模糊"
2. 录制过程中保持模糊
3. 录制完成后关闭

### 公共场所使用

在公共场所使用时：

1. 启用隐私模式
2. 防止他人看到敏感内容
3. 保护个人隐私

### 远程协助

远程协助或屏幕共享时：

1. 提前启用隐私模式
2. 共享屏幕
3. 避免泄露隐私信息

## 高级功能

### 白名单

设置不需要模糊的页面：

1. 添加白名单 URL
2. 这些页面不会应用模糊
3. 适合信任的环境

### 黑名单

设置必须模糊的页面：

1. 添加黑名单 URL
2. 这些页面强制模糊
3. 即使关闭隐私模式也生效

### 定时模糊

设置定时自动模糊：

1. 配置时间段
2. 在指定时间自动启用模糊
3. 适合固定时间使用

### 模糊样式自定义

自定义模糊效果样式：

1. 选择模糊方式：
   - **高斯模糊**: 标准模糊效果
   - **马赛克**: 像素化效果
   - **纯色遮罩**: 用纯色覆盖
   - **自定义**: 使用自定义 CSS

2. 配置样式参数
3. 实时预览效果

## 使用技巧

### 快速切换

- 使用快捷键快速切换模糊状态
- 截图前启用，截图后关闭
- 养成使用习惯

### 合理配置

- 根据使用场景选择保护级别
- 不要过度模糊影响使用
- 平衡隐私和便利性

### 测试效果

- 启用模糊后先测试截图
- 确认敏感信息已被保护
- 调整模糊强度到合适程度

### 多场景配置

- 为不同场景创建配置
- 快速切换配置
- 提高使用效率

## 隐私建议

### 基本原则

1. **最小化原则**: 只分享必要的信息
2. **模糊原则**: 不确定时选择模糊
3. **检查原则**: 分享前仔细检查

### 分享前检查

分享截图或视频前检查：

- ✅ 封面是否模糊
- ✅ 标题是否模糊
- ✅ 番号是否模糊
- ✅ 个人信息是否移除
- ✅ 浏览器标签是否关闭

### 其他隐私措施

除了使用隐私保护功能，还应该：

1. **使用隐私浏览模式**: 不保存历史记录
2. **定期清理数据**: 清除浏览历史和缓存
3. **不在公共电脑使用**: 避免在公共电脑上使用
4. **使用强密码**: 保护账号安全
5. **启用两步验证**: 增强账号安全

## 故障排查

### 模糊不生效

检查：

1. 是否启用了隐私保护
2. 是否按了快捷键
3. 刷新页面重试
4. 查看浏览器控制台错误

### 模糊效果不理想

调整：

1. 增加模糊强度
2. 更改模糊方式
3. 添加自定义保护区域
4. 调整保护级别

### 快捷键冲突

解决：

1. 更改为其他快捷键
2. 使用组合键（Ctrl+Shift+X）
3. 禁用冲突的扩展

### 影响页面性能

优化：

1. 降低模糊强度
2. 减少保护区域
3. 只在需要时启用
4. 关闭自动模糊

## 常见问题

### 隐私模式会保存数据吗？

不会，隐私模式只是视觉效果，不会修改或保存任何数据。

### 可以永久启用吗？

可以，在设置中开启"默认启用模糊"。

### 会影响扩展其他功能吗？

不会，隐私模式只影响显示效果，不影响功能。

### 可以自定义模糊样式吗？

可以，在高级设置中自定义 CSS 样式。
